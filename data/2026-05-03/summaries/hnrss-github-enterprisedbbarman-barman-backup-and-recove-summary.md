# Barman – Backup and Recovery Manager for PostgreSQL (GitHub)

## Overview
- Open‑source administration tool for disaster recovery of PostgreSQL servers, written in Python.  
- Enables remote backups of multiple servers in business‑critical environments, reducing risk and assisting DBAs during recovery.  
- Hosted by EnterpriseDB; replaces the legacy SourceForge repository starting with version 2.13.
