# Claude Code's New Routines Feature Works Offline on Macs

## What are Claude Code routines?

Claude Code is a software application part of the Claude app for Mac. It allows users to automate tasks by scheduling repeating actions based on user input.

## Anthropic Releases Claude Code with New Offline Support and Enhanced Features

Anthropic, a developer of automated tools, recently added a new feature to their software suite called "Claude Code's Routines." This update enables automated tasks that can run without the need for an internet connection. The updates also include significant redesigns to improve user experience and productivity.

## Key Benefits of Offline Support:

- Users can schedule routines to run on-demand, saving time and resources.

## Additional Features Include:
 
- Integration with multiple Mac operating systems: 
  - Pro (5 automation limits)
  - Max (15 automation limits)
  - Team/Enterprise (25 automation limits)

## Enhanced Productivity Tools

- Parallel sessions: Multiple Claude sessions can be run side by side from a single window.
- Integrated terminal, file editing, HTML, PDF preview, and drag-and-drop layout for easier workflow optimization.

    Additionally the redesigned Claude Code experience includes:
 
        - Faster diff viewer
        - All features optimized for Mac usability

The redesign of Claude Code's app with parallel sessions and integrated tools provides users with a comprehensive solution for automated workflows.
