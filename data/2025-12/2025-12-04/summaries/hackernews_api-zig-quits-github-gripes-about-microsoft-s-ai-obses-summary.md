# Zig Programming Language Foundation Abandons GitHub Commitment to Engineering Excellence

The Zig programming language foundation has quit GitHub due to criticism of the code-sharing site's decline. The decision comes after a thread on the platform, titled "safe_sleep.sh rarely hangs indefinitely," exposed significant problems with an existing bug.

**Background**

In April 2025, a user named AlekseiNikiforovIBM posted a problem that involved a Safe Sleep script failing to work as intended in GitHub Actions. The issue persisted for many months until August 2023, when the code was addressed and left open in an unresolved thread. This prolonged exposure led Zig core developer Matthew Lugg to comment on the obvious issues with the script.

**Impact of the Problem**

The Safe Sleep script failed to pause execution safely even under normal conditions due to incorrect assumptions about the system time. The failure caused an issue on Zig's CI runner machines, causing multiple processes to take down services for weeks.

Meanwhile, Microsoft's AI obsession appears to have contributed to the decline of GitHub as a whole. In August 2023, the company announced that it was moving its operations to Codeberg and abandoned its commitment to engineering excellence on the platform.
