**Advent of AI Series: Goose - An Open- source, Extensible AI Agent**

### About Goose

The Advent of AI series leverages Goose, an open-source, extensible AI agent that enables automating complex engineering tasks from start to finish.

**Key Features**

* Automates development tasks
* Builds entire projects from scratch
* Writes and executes code
* Debugs failures
* Orchestrates workflows
* Interacts with external APIs

### Using Goose for Complex Development Tasks

Goose is designed to work with any Large Language Model (LLM) and supports multi-model configuration to optimize performance and cost.

### The Homecoming Board Project - A Gesture-Controlled Flight Arrival Display

Using Goose, a gesture-controlled flight arrival display was built with hand gestures. No screens were touched in the freezing cold.

** Tech Stack**

This project utilized React + TypeScript with SSR (TanStack Start), MediaPipe for gesture recognition, and the OpenSky Network API for real-time flight data.

### Goose in Action - Demo Application

The demo application showcases how to build secure MCP servers and clients using contextual access policies. A minimal chat application demonstrates remote Model Context Protocol server security.
