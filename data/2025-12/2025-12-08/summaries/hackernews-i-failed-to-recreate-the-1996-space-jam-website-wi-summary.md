# I failed to recreate the 1996 Space Jam Website with Claude | j0nah.com

## Failed Attempt to Replicate Space Jam Website

The article discusses an unsuccessful effort by Claude (an AI model) to recreate the 1996 Space Jam website using only a screenshot. The goal was to preserve the original website forever and replicate it from scratch. Despite having all assets and screenshots, Claude failed to create an accurate replica with the specified requirements.

## Setup Preparation

Before recreating the site, Claude received:

* A screenshot of the original 1996 Space Jam landing page
* Raw image assets (specifically a tiling starfield GIF background)

To aid in their interaction, Claude used a man-in-the-middle proxy to capture and log user prompts, API calls (e.g., reading, writing files), and tool invocations.

## Part 1: Initial Observe

The original Space Jam website features:

* A single HTML page with absolute positioning of every component
* An exact tiling starfield GIF background
* Assets and elements positioned using tables in the original site

Claude had all these assets, but when asked to recreate the landing page exactly as per the screenshot:

- Claude produced a satisfactory replica with some adjustments
However, the overall layout deviated from the original 1996 vision.

## Key Findings

Claude achieved two main objectives:
1. Studied and analyzed details of the original design (orbital patterns, spacing relationships)
2. Applied insights to recreate the site in his own image


Note that Claude's replica still has some deviations, primarily because it doesn't accurately represent the dynamic behavior of the original 1996 website.
