# Tiny Core Linux, Micro Core Linux, 12MB Linux GUI Desktop, Live, Frugal, Extendable

#### Welcome to Tiny Core Linux
Tiny Core Linux is a highly modular system with community build extensions that starts with a recent Linux kernel and advanced root filesystem components.

##### Key Features of Tiny Core
* Starts with kernel vmlinuz and root filesystem packages in core.gz
* A simplified desktop platform with user-specified applications and hardware support via included extensions

##### TinyCore Overview
TinyCore is an example of what the Core Project can produce, offering a 16MB FLTK/FLWM desktop. It provides an easy-to-set-up way to create basic desktop environments using community packages.

##### Design Principles
* Simple framework for creating custom desktop environments
* Includes community-packaged applications and hardware support
* Allows users to install additional software via online repositories or compilation

##### Current Version
The latest version is 16.2 (update available)

##### About the Project
Tiny Core Linux aims to create a nomadic, ultra-small graphical operating system that can boot from removable media, such as CD/dvdROM, pendrives, or hard drives. The desktop runs quickly and supports additional applications and hardware of the user's choice.

##### Community Involvement
The Tiny Core project offers an open development model with active forums for discussion, knowledge sharing, and contributions. Anyone can participate in building extensions by contributing their favorite application or hardware support to run on Tiny Core.
