# Pricing Changes for GitHub Actions

**Background**

* On January 1, 2026, GitHub will start charging $0.002 per-minute for self-hosted runners.
* The change aims to realign costs with usage and value delivered to every Actions user.

## Key Points

* **Free for Public Repositories**: Self-hosted runner customers can continue using the platform free of charge.
* **88% of Customers Won't Notice a Price Increase**
	+ 96% will see no cost increase or decrease in their bills.
	+ Only 4% of users, primarily individuals and small teams, will experience a moderate price increase ($13).

## Impact

* **88% Customers Won't See a Price Change**: Self-hosting runners can maintain free usage on public repositories.
* **85% of Users in 4 Cohort Will Experience a Decrease**: Only those impacted by this change will see their total Actions bill decrease ($13) or median increase.
