# Building a 3D Virtual Portfolio Room🏠 - DEV Community

## Building a 3D Interactive Portfolio Room

### What is this Project?

A0D3 interactive room portfolio built using HTML, CSS, and JavaScript, showcasing certificates as art in a gallery-like setting.

### Technical Stack

* Used CSS 3D transforms for dynamic effects
* Ambient sound effects and animations (e.g., an opening door)
* Keyboard interactions (+/+, zooming)
* Responsive design on both desktop and mobile
* Licensed under MIT for open collaboration

### What's Still Missing
