# Building a Stunning Travel Agency Landing Page with Uno Platform — My WOW Factor Challenge Submission 🤩 - DEV Community

**Building a Stunning Travel Agency Landing Page with Uno Platform - WOW Factor Submission**

After completing my first project for the Uno Platform Hot Design Challenge, I realized how much I enjoyed the process and decided to dive back in.

**The Unexpected Journey**

When I went back to the challenge and read more about "WOW Factor", it felt like an invitation I couldn't ignore. Instead of a mobile UI, this time I decided to build a modern, clean travel agency landing page entirely powered by Uno Platform.

**A More Immersive Experience**

This project had a bigger scale and more features than my previous challenge:

* Full-width imagery
* Elegant typography
* Destination cards with mock data
* Animations
* Confetti celebration subscription section for an immersive experience

**Why I Built It**

The "WOW Factor" prompt asked for:

* A visually engaging home screen
* A fun theme (like a coffee shop)
* Creative visuals, mock data, animations
* Smooth performance
* Something that grabs attention immediately

My goal was to create a landing page that would make the judges say "WOW" at least once.

**Design Vision**

For this project:

* Create a hero section with bold visuals and a layered layout
* Use a welcoming confetti animation after subscribing
* Add smooth vertical scrolling for immersive storytelling
* Destination cards with mock data, rounded corners, shadows, and responsive images
* Subtle animations to create delight without slowing performance
