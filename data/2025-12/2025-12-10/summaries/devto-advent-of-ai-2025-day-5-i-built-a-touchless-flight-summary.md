# Advent of AI 2025 - Day 5: I Built a Touchless Flight Tracker You Control With Hand Gestures - DEV Community

### Adapting Goose: A Gesture-Controlled Flight Tracker

The Advent of AIseries leverages the Goose open source AI agent to automate engineering tasks, providing a robust platform for developers.

#### Key Points:

* Goose is an extensible AI agent that can be used for complex development tasks from start to finish.
* It offers precision execution and supports multi-model configuration for optimal performance and cost.
* Goose works with any Large Language Model (LLM) and seamlessly integrates with MCP servers.
* The platform is available as both a desktop app and CLI for flexible integration.

### Adapting Goose for The Homecoming Board Project

For this gesture-controlled flight arrival display, I built "The Homecoming Board" using the Goose AI agent. This project involved:

1. Real-time flight data integration via the OpenSky Network API.
2. Audio feedback for gesture recognition.
3. Navigation using at least two distinct gestures.

#### Tech Stack Used:

* TanStack Start (React + TypeScript with SSR) for building The Homecoming Board app.
* MediaPipe for gesture recognition.
* TheOpenSky Network API for real-time flight data.

**TLDR;**

* Built "The Homecoming Board" with Goose AI agent, integrating real-time flight data and audio feedback via the OpenSky Network API.
* Developed using TanStack Start, MediaPipe for gesture recognition, and MCP servers.
