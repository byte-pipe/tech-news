# I can't upgrade to Windows 11, now leave me alone

## Summary

The article discusses Microsoft's decision not to support Windows 11 due to hardware constraints, specifically requiring a Trusted Platform Module (TPM 2.0). However, Microsoft continues to harass users with "reminder" messages, offering alternatives and ads for new laptops. The author argues that this is intentional design, rather than an error or bug.

## Key Points

* Microsoft won't support Windows 11 due to hardware requirements
* Hardware limitation is specifically TPM 2.0
* Users are prevented from updating due to TPMS requirement
* Notification messages are "reminder" instead of clear updates
* Offers alternatives and ads for new laptops, asking users to create a Microsoft account
* User's device supports Windows 11 but not explicitly supported by MS

## Dismissal Mechanism Overview

To dismiss the notification, the user can:

- Ignore reminders by moving on with daily activities
- Click "remind me later," allowing Microsoft to show the message again in the future
- Click "learn more" and be shown ads for new laptops, which may encourage the user to create a Microsoft account
