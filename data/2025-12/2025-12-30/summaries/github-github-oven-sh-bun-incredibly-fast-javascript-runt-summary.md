# GitHub - oven-sh/bun: Incredibly fast JavaScript runtime, bundler, test runner, and package manager – all in one

## oven-sh/bun: Fast JavaScript Runtime in One Package

### Overview
The `oven-sh/bin/bun` package offers an incredibly fast JavaScript runtime, bundler, test runner, and package manager. It includes all the necessary tools for efficient development and deployment.

### Features
- Incredibly fast execution speed
- Bundler: compiles and packages code into a single binary
- Test Runner: runs tests with confidence guarantees
- Package Manager: easily installs and upgrades dependencies

## Getting Started with oven-sh/bun
Here's a simple example of how to run tests and compile JavaScript code:
```bash
npx test-bun myscript.js
```
This will execute the `myscript.js` file using bun and report any compilation errors or warnings.
