# GitHub - shiyu-coder/Kronos: Kronos: A Foundation Model for the Language of Financial Markets · GitHub

Kronos
================

Overview
--------

Kronos is an open-source foundation model designed for financial markets, specifically for training and predicting K-line sequences.

Features
--------

* Trained on data from over 45 global exchanges
* A second-stage framework with a specialized tokenizer and transformer pre-training
* Hierarchical discrete tokens processing
* Two-stage architecture for handling high-noise characteristics of financial data

Live Demo
--------

A live demo is available showcasing forecast results for the BTC/USDT trading pair.

Model Zoo
---------

Kronos provides a range of pre-trained models with varying capacities to suit different needs and applications. Models are readily accessible from the Hugging Face Hub.

Key Points
------------

* Open-source foundation model designed for financial markets and K-line sequences
* Specialized tokenizer and transformer pre-training for high-noise character analysis
* Hierarchical discrete token processing
* Two-stage architecture with a focus on forecast generation
