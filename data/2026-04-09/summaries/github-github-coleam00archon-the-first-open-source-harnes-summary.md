**Overview**

Archon is an open-source workflow engine designed for AI coding agents, aiming to make deterministic and repeatable development processes.

**Key Features**

* **Deterministic and Repeatable**: Define development workflows as YAML files and run them reliably across multiple projects.
* **Isolated Combinatorics**: Each workflow run gets its own git worktree, eliminating conflicts.
* **Fire-and-Forget**: Launch a workflow while working on other tasks, then return to a finished PR with review comments.

**Benefits**

* Reduced developer time and effort
* Improved reliability and consistency
* Enhanced collaboration through standardized workflows

**Setup and Use**

* Install the Archon repository files (workflows.yaml) as part of your development process.
* Configure the engine to run workflows on schedule using Git hooks or environment variables.
