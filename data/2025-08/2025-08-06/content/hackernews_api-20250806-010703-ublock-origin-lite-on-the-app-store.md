## Description

uBO Lite (uBOL) is a reliable and efficient content blocker.The default ruleset corresponds to uBlock Origin's default filterset:- uBlock Origin's built-in filter lists- EasyList- EasyPrivacy- Peter Lowe’s Ad and tracking server listYou can enable more rulesets by visiting the options page -- click the _Cogs_ icon in the popup panel.uBOL is entirely declarative, meaning there is no need for a permanent uBOL process for the filtering to occur, and CSS/JS injection-based content filtering is performed reliably by the browser itself rather than by the extension. This means that uBOL itself does not consume CPU/memory resources while content blocking is ongoing -- uBOL's service worker process is required _only_ when you interact with the popup panel or the option pages.
