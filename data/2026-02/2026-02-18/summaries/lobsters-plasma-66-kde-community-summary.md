# Plasma 6.6
## A visual guide to the best desktop in the known universe (according to us)

Plasma 6.6 focuses on enhancing usability and accessibility while maintaining its flexibility and features. Key improvements include a new on-screen keyboard, Spectacle text recognition, and a revamped Plasma Setup wizard for initial user account configuration.

## Highlights
### On-Screen Keyboard
A new and improved on-screen keyboard is available.
### Spectacle Text Recognition
Spectacle can now extract text from images in screenshots.
### Plasma Setup
A dedicated wizard for setting up user accounts after OS installation.

## New Features
Users can now create custom global themes for their environment, impacting both day and night themes. App appearance can be subtly modified by adjusting color intensity. A new skin tone selector simplifies emoji skin tone choices. Quick Wi-Fi connection via QR code scanning is now possible. Volume adjustment for playing apps is available in the task manager. Enabling "Open on hover" in the Windows Listwidget and filtering windows by desktop/activity offer workflow improvements. Double-clicking files/folders on the desktop with Alt key provides properties.

## Accessibility
Enhanced accessibility features include four grayscale color blindness filters in System Settings. The Zoom and Magnifier tools now have a centered pointer tracking mode. Support for "Slow Keys" on Wayland and a standardized "Reduced Motion" accessibility setting have been added.

## Screenshots and Screen Recording
Spectacle can recognize and extract text from images, beneficial for visually impaired users. Users can filter windows out of screencasts via a right-click menu.

## Virtual Keyboard
A new Plasma Keyboard is introduced.

## Plasma Setup
The new Plasma Setup wizard separates user account setup from the operating system installation, useful for pre-installed systems, refurbished computers, and secure computer distribution.

## But That’s Not All…
Additional features include virtual desktops on the primary screen only, an optional new login manager, optional automatic screen brightness, optional game controller input, font installation in the Discover software center, process priority selection in System Monitor, pinned standalone Web Browser and Audio Volume widgets, USB access prompts, visual refresh of permission prompts, and smoother animations on high-refresh-rate screens.

## In Memory of Björn Balazs
This release is dedicated to Björn Balazs, a passionate contributor who passed away. His dedication to user privacy and control over technology is highly valued by the KDE community.
