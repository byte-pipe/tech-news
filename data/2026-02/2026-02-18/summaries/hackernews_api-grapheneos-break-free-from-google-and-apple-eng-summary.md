## What is GrapheneOS
GrapheneOS is a custom, open-source Android operating system prioritizing user privacy and security. It's built upon the Android Open Source Project (AOSP) but significantly differs from standard Android by completely eliminating Google service integration. This prevents corporate tracking and data collection while maintaining a modern and stable environment. The system features enhanced kernel and component hardening for security and allows running Google Play Services in a sandboxed environment for app usage without broad system permissions. Currently, it primarily supports Google Pixel devices, leveraging their Titan M security chips for data protection.

## Dedicated devices
Initially supporting multiple manufacturers, GrapheneOS now primarily focuses on Google Pixel devices. While it might be possible to run on other devices, it's not officially guaranteed and requires potential porting. The author notes the irony of a privacy-focused system being best suited for Google's own devices, highlighting features like Verified Boot, Titan M, IOMMU, and MTE as key reasons.
