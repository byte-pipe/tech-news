# Maple Mono: Open source monospace font

Maple Mono is a monospace font designed for programmers, aiming to enhance coding productivity with subtle elegance and readability. It addresses shortcomings found in other popular monospace fonts like JetBrains Mono, Fira Code, and Victor Mono by offering features such as round corners, a cursive-style italic, fine-grained customization, built-in Nerd-Font support, and a 2:1 monospace ratio for Chinese and English.

## Key Features

* **Variable Font Support:** Offers a range of weights for flexible styling.
* **Distinct Glyphs & Cursive Italics:** Features new glyph designs with a unique cursive italic style for enhanced visual appeal.
* **Smart Ligatures:** Includes plain text tags and connected letter ligatures for improved code readability and aesthetics.
* **Beautiful Icons (Nerd-Font):** Provides first-class support for Nerd-Font, enabling the use of icons in terminals.
* **Customizable Features:** Users can configure or freeze OpenType features to tailor the font to their preferences.
