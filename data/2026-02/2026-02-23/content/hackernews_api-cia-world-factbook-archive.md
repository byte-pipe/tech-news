About This Project

 All data originates from the CIA World Factbook, a public-domain U.S. Government
 publication. This project is not affiliated with the Central Intelligence Agency or
 the U.S. Government. Analysis follows ICD 203 analytic standards. Data spans
 36 editions (1990–2025)
 covering 281 entities with
 1,061,341 parsed data fields.
