# EDuke32: A Powerful Duke Nukem 3D Port

EDuke32 is a free, open-source game engine and source port of the classic PC first-person shooter Duke Nukem 3D. It is compatible with Windows, Linux, macOS, FreeBSD, and various handheld devices. The engine includes numerous enhancements, features, and scripting extensions for both regular players and mod developers.

## Key Features

* **Native Operation:** Runs without emulation on modern Windows versions (11, 10, 8, 7) and is well-supported on Linux (native SDL or Wine).
* **High Resolution Support:** Capable of rendering at very high screen resolutions (e.g., 10240x4320).
* **Rendering Options:** Offers a choice between hardware-accelerated OpenGL renderers and the classic software rendering mode.
* **Bug Fixes:** Addresses numerous programming errors present in the original DOS version, improving stability.
* **Voicewarrior Port:** Includes a fully functional port of Shadow Warrior with all its features.
* **Active Development:** The only Duke3D port actively developed and maintained for over two decades.
* **Polymer Renderer:** Features an advanced, hardware-accelerated rendering engine with dynamic lighting, shadows, specular/normal map support, various texture formats, fog density, full brights and glow maps, detail textures, animation blending, colored fog, and extensive screen customization options.
* **Enhanced Scripting:** Provides a vast number of new extensions to the game's scripting system, enabling complex gameplay modifications.
* **HRP Support:** Runs the HRP (a popular modding tool) with all its features.
* **Full-Featured Console:** Includes a Quake-style console with key bindings, command aliases, history, and colored text.
* **Code Rewrites and Optimizations:** Incorporates hundreds of code improvements and bug fixes.
* **User-Friendly Features:** Adds modern status displays, mod loading from the startup window, and WASD-based controls with improved mouse aiming.
* **Sound Support:** Supports Ogg Vorbis and FLAC audio formats.
