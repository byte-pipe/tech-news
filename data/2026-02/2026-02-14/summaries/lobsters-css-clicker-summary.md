# CSS Clicker

# CSS Clicker Summary

* **Overview:** CSS Clicker is an idle game built entirely in HTML/CSS, requiring no JavaScript or server-side code. The objective is to build and upgrade a website to become the most famous on the internet.
* **Gameplay:** Players earn money by clicking, which is used to purchase various upgrades for their website, such as animating the views counter, enabling monitor glow, and showing cursor icons.
* **Saving Progress:** Progress is not automatically saved. Players must avoid closing or refreshing the tab to retain their progress. Ctrl+Shift+R can be used to refresh without this behavior.
* **Sharing the Game:** Users can share the game by taking a screenshot and sending it via Mastodon, Twitter, or Orbsky, or by emailing the developer. A high-quality PDF can also be saved using Ctrl+P, ensuring "Background graphics" are enabled in print settings.
* **Technical Aspects:** The game is fully functional even with JavaScript disabled. The developer uses "scripting media querylol" to detect if JavaScript is disabled. The codebase is hand-written and available for viewing (right-click "view source") and offline saving (Ctrl+S), as well as on GitHub.
* **Mobile Compatibility:** The game is surprisingly playable on mobile devices with Chrome or Firefox on Android, though it was primarily designed for desktop browsers and may have issues on Safari.
* **Motivation for Development:** The developer created the game as a fun technical challenge, inspired by the CSS crimes platform, and to challenge the notion that HTML/CSS are not programming languages.
* **Usage of Assets:** Users are welcome to take ideas and assets from the game for their own website projects.
* **Credits:** The game features intro music by Femtanyl, cursors, GIFs, and badges from various sources (primarily The 88x31 GIF Collection), original art by oneko, playtesting by Mike Klubnika, background photos by the developer, and inspiration from the CSS crimes community.
