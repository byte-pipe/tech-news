# Grok2API: A FastAPI-based API for Grok2

- This GitHub repository, chenyme/grok2api, provides a Grok2 API rebuilt using FastAPI.
- It fully supports the latest Web calling format for Grok2.
- Key features include support for both streaming and non-streaming conversations.
- It enables image generation and editing capabilities.
- The API supports "deep thinking" functionalities.
- It integrates a pool of instances with automatic load balancing.
- The project is licensed under the MIT license.
- It has received 1.4k stars and 434 forks.
- A live demo is available at grok2api-omega.vercel.app.
