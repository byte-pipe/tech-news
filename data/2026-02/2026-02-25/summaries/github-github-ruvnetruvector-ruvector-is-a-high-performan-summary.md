# RuVector: A High-Performance Vector and Graph Database

- RuVector is a vector and graph database developed in Rust.
- It is designed for applications in AI, agentic systems, and real-time analytics.
- The database combines several key technologies:
    - HNSW search for efficient similarity search.
    - Dynamic minimum cut coherence for graph intelligence.
    - Self-learning memory for enhanced reasoning.
- These components are integrated into a unified engine to enable scalable and low-latency reasoning and structured data retrieval.
- The project is licensed under the MIT license and has received 642 stars and 122 forks on GitHub.
- Further information can be found at ruv.io.
