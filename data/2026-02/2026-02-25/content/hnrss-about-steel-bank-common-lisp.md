# Steel Bank Common Lisp

# About

Steel Bank Common Lisp (SBCL) is a high performance Common Lisp
compiler. It is open source / free software, with a permissive license. In
addition to the compiler and runtime system for ANSI Common Lisp, it provides
an interactive environment including a debugger, a statistical profiler, a
code coverage tool, and many other extensions.

SBCL runs on Linux, various BSDs, macOS, Solaris, and Windows. See thedownloadpage for supported platforms, andgetting startedguide for additional help.

The most recent version isSBCL 2.6.1, released January 26, 2026 (release notes).

## Documentation

SBCL's manual is available on the web inhtmlandpdfformats. See thedoc/manualdirectory in the source code for the current version in TeXInfo source.

## Reporting Bugs

Bugs can either be reported directly toSBCL's bug database on Launchpad, or by sending email to thesbcl-bugs@lists.sourceforge.netmailing list -- no subscription required.
