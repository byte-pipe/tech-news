# Offpunk 3.0 "A Community is Born" Release

This article by Ploum announces the release of Offpunk 3.0, an offline-first command-line browser. The development of this version has become a collaborative effort, with significant contributions from the community in areas like translation and packaging. The author expresses gratitude for these contributions and highlights the growing Offpunk community.

## Key Changes in Offpunk 3.0

* **Language Support:** Offpunk is now translatable and includes translations for Spanish, Galician, and Dutch.
* **`openk` as a Standalone Tool:** The `opnk` standalone tool has been renamed to `openk`, which opens files in the terminal or in the user's preferred software.
* **`xkcdpunk` for XKCD Comics:** A new standalone tool, `xkcdpunk`, displays XKCD comics directly in the terminal.
* **`unmerdify` for Content Extraction:** Integration of `unmerdify`, a library that extracts content from HTML articles using FiveFilters rules, is included. Users can contribute by creating or improving these rules.
* **Social Features:** New "share" (email URL) and "reply" (email author) functionalities have been added.
* **Browser with Login Capabilities:** The "cookies" command allows importing cookie files for accessing websites requiring login.
* **Improved Image Handling:** Images are displayed by default in Gemini with increased size, configurable in the `offpunkrc` file.
* **Display Hidden RSS/Atom Links:** Hidden RSS/Atom feed links are now displayed on HTML pages.
* **Display Blocked Links:** Links to blocked domains are displayed in red by default, configurable in the `offpunkrc` file.
* **Preset Themes:** Support for multiple themes ("offpunk1", "cyan", "yellow", "bw") is introduced.
* **Enhanced Redirects and Blocking:** Redirects now operate at the netcache level, and blocking is more robust.
* **Other Improvements:** Includes a smarter "root" command, a new `websearch` command using wiby.me, a customizable default command, a view switch, help via email, bug reporting, and various bug fixes.

The author encourages users to report any new bugs and to join the growing Offpunk community.
