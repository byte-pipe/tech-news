# Unstract: No-code LLM Platform for Structuring Unstructured Documents

- Unstract is a no-code platform designed to simplify the process of launching APIs and building ETL (Extract, Transform, Load) pipelines for structuring unstructured documents.
- It leverages Large Language Models (LLMs) to automatically extract information and structure data from various document types.
- The platform aims to eliminate the need for extensive coding, making it accessible to users without deep technical expertise.
- Unstract is released under the AGPL-3.0 license.
- It has garnered significant interest within the developer community, with 6.2k stars and 595 forks on GitHub.
- More information can be found on the official website: unstract.com.
