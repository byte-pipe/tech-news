# OpenSandbox Overview
=====================

## General Purpose Sandbox Platform for AI Applications

OpenSandbox is a general-purpose sandbox platform designed specifically for AI applications. The platform offers multi-language SDKs, unified sandbox APIs, and Docker/Kubernetes runtimes for various scenarios such as:

*   Coding Agents
*   GUI Agents
*   Agent Evaluation
*   AI Code Execution
*   RL Training
