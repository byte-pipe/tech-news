# Flashpoint Archive Summary

*   Flashpoint Archive is a community-driven project dedicated to preserving games and animations from the web, recognizing the importance of internet history and culture.
*   Since December 2017, the archive has preserved over 200,000 games and animations across more than 100 browser plugins and web technologies.
*   The project provides a flexible, open-source software package for reliable navigation and playback of preserved content. This includes a launcher, proxy, and sandbox.
*   Initially started by BlueMaxima to address the disappearance of web games due to the demise of Flash, Flashpoint Archive has grown into a large-scale effort with numerous community contributors.
*   Flashpoint Archive operates as a non-profit organization focused on the preservation and accessibility of digital interactive experiences from the web. Support can be provided through donations via the website.
*   The website offers options to search the collection and request content for archival.
