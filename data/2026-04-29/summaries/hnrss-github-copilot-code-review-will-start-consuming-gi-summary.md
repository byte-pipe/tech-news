# GitHub Copilot code review will start consuming GitHub Actions minutes on June 1, 2026 - GitHub Changelog

# GitHub Copilot code review billing change – summary

## What’s changing
- Starting June 1 2026, each Copilot code review will be billed in two ways:
  - All Copilot usage (including code reviews) is charged as AI Credits under the new usage‑based billing model.
  - GitHub Actions minutes are consumed from your existing plan entitlement for reviews on private repositories; any excess minutes are billed at standard GitHub Actions rates.
- Public repositories remain unaffected – Actions minutes stay free.
- The change applies to the following plans:
  - GitHub Copilot Pro
  - GitHub Copilot Pro+
  - GitHub Copilot Business
  - GitHub Copilot Enterprise
- Includes code reviews run by non‑licensed users and those billed via direct organization billing.

## When it takes effect
- The new billing model becomes active on **June 1 2026**.
- Until that date, Copilot code review usage continues to draw only from the existing Copilot premium request unit (PRU) allowance and does not consume Actions minutes.

## What you need to do
### Review billing and usage
- Check current GitHub Actions minute consumption in your account or organization billing settings.
- Verify that your personal or organizational Actions spending budget aligns with expected usage; adjust limits if needed.
- Monitor Copilot and Actions usage via Copilot usage metrics, Actions metrics, and the Billing Usage Report.
- Read the usage‑based billing announcement to understand how Copilot usage will be measured.
- Inform billing administrators and engineering leads about the upcoming change.

### Review runner settings
- No extra setup is required if GitHub‑hosted runners are already enabled.
- To customize the GitHub‑hosted runner environment, consider upgrading to larger runners.
- Self‑hosted runner configuration remains available.

For more details, see the documentation and join the discussion in the GitHub Community.
