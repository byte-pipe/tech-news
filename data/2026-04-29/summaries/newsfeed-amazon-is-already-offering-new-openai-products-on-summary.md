# Amazon is already offering new OpenAI products on AWS

## Key developments
- After OpenAI announced a revised agreement with Microsoft that ends Microsoft’s exclusivity, Amazon highlighted the change as “very interesting.”
- AWS added OpenAI’s latest models, the Codex code‑writing service, and a new “Bedrock Managed Agents” product to its Bedrock platform.
- Bedrock Managed Agents is built to run OpenAI reasoning models, providing agent steering and security features.
- Amazon’s blog frames the launch as the start of a “deeper collaboration” between AWS and OpenAI.

## Strategic context
- The updated OpenAI/Microsoft deal resolves a prior limitation that prevented AWS from offering OpenAI products, despite a reported up‑to‑$50 billion deal between OpenAI and Amazon.
- Tensions have been reported between Microsoft and OpenAI, with both parties exploring partnerships with rivals: OpenAI with AWS and Oracle; Microsoft with Anthropic and its own Claude‑based agent offering.

## Industry implications
- The move positions AWS as a direct competitor to Microsoft’s Azure for hosting OpenAI services.
- It signals a broader shift in the AI cloud market, where major providers are diversifying partnerships to reduce reliance on a single vendor.
