# Linux Foundation: The "Cockpit" of the Cloud

## Introduction to Linux and the Linux File Hierarchy System (FHS)

* Linux was first developed as an open-source operating system by Linus Torvalds in 1991.
* It is now widely used on servers, supercomputers, embedded systems, and other devices due to its reliability, security, and flexibility.

## The Role of DevOps in Linux

* DevOps is a concept that emphasizes integrating the development and operations phases of software development.
* In Linux, the DevOps model relies on a strong understanding of the file hierarchy system (FHS) and shell commands.

## Redirection & Pipes: The "Lego" Logic of Linux

* Writing programs in Linux is based on the core philosophy of "write one thing well and work together."
* Piping allows two separate commands to output their combined results to a single destination.
* Using redirects (e.g., `>`, `>>`) enables writing directly to a file.
