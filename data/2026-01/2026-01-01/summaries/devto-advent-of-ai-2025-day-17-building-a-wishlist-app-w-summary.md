# Advent of AI 2025 - Day 17: Building a Wishlist App with Goose and MCP-UI - DEV Community

## Day 17: Building a Wishlist App with Goose and MCP-UI

**Introduction**

In this article, we built a wishlist app using the MCP-UI framework and an open source AI agent called Goose. This app is designed to work seamlessly with any LLM (Language Model Language), making it a flexible and efficient tool for developers.

** goose Overview**

Goose is an open-source AI agent that provides features such as code suggestions, editing, testing, and more. It can automatically generate entire projects from scratch, including development tasks from start to finish.

**Winter Wishlist App with Goose and MCP-UI**

This article focuses on building a wishlist app for the Advent of AI series using MCP-UI and Goose. We created a user-friendly interface that allows users to:

* Add wishes with a quick message
* View all their wishes in a nicely formatted UI
* Grant wishes when they come true
* Delete wishes they no longer want

**Key Features**

* Make Wishes: Users can input their wish, categorized and prioritized.
* View Wishes: The app displays the list of wishes with descriptions and priorities.
* Grant Wishes: Users can tap "Confirm" to grant their wishes, and then choose when they come true.
* Release Wishes: Users can tap "Cancel" to delete a wishing process.

**Benefits**

This wishlist app offers several benefits for users:

* Streamlined wish management
* Quick addition of new wishes
* Easy tracking and deletion of wishes

By leveraging Goose's AI features, this app has maximized flexibility and efficiency. With MCP-UI integration, we were able to create an intuitive visual interface that adapts to the user's workflow. Our winter wishlist app is now ready for deployment!
