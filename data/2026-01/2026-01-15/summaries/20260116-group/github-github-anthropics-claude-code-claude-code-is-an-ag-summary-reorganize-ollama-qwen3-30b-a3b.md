# Claude Code - Terminal-Based AI Coding Assistant

## Overview
Claude Code is an agentic coding tool designed to operate directly within the user's terminal. It comprehends the user's codebase and accelerates development through natural language interaction.

## Core Features
- **Routine task automation**: Executes repetitive coding tasks via natural language commands
- **Codebase explanation**: Provides clear insights into complex code structures
- **Git workflow management**: Handles version control operations using conversational prompts
- **Terminal-native operation**: Functions entirely within terminal environment without requiring browser access
