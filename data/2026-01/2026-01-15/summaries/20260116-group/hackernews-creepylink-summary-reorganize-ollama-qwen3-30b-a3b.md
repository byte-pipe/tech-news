# CreepyLink

## Overview
CreepyLink is a novel link shortener that subtly alters trusted URLs to create misleading, suspicious-looking links. The service aims to increase user hesitation and curiosity before clicking, promoting safer online behavior.

## Key Features
- **Reduced Credibility**
  Transforms legitimate URLs into visually suspicious formats, making users question authenticity.
- **Subtle Modifications**
  Applies minor, intentional changes to original URLs (e.g., misspellings, added characters) without altering functionality.
- **Suspicious Appearance**
  Converts familiar, trusted links into deceptive representations that trigger cautious user behavior.

## Purpose
The service addresses impulsive clicking by introducing psychological friction, encouraging users to verify links before proceeding. This approach leverages natural curiosity and suspicion to improve online safety awareness.
