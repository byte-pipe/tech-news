# usememos: An Open-Source Self-Hosted Note-Taking Service

## Overview
usememos is an open-source, self-hosted note-taking platform designed to give users complete ownership of their thoughts and data. It operates without tracking, advertisements, or subscriptions, ensuring full user control.

## Key Features

* **No Tracking or Ads**: Complete privacy with zero data monetization or external tracking.
* **Anonymous Interaction**: Adjust notification settings without revealing personal identity.
* **Self-Hosted Control**: Host the service on your own infrastructure for full data sovereignty.
* **Open-Source Transparency**: All code is freely available for inspection and modification.
