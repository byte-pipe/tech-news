# GitHub - nocodb/nocodb: 🔥 🔥 🔥 Open Source Airtable Alternative

** nocodb Overview **
=====================

The nitrocloud office, also known as `nocodb.com`, is an open-source alternative to Airtable. It offers a robust platform for managing data, with features such as tables, forms, and views.

** Key Features **

*   Open-source under the AGPL-3.0 license
*   Supports up to 61.1k columns and 4.5k records per table
*   Forked by over 11,000 developers on GitHub

** How it Works **
=====================

-nocodb provides a web-based interface for creating, managing, and sharing data across various platforms.

-   Users can create tables with custom fields and relationships.
-   Users can add forms and views to the table to further customize their workflows.
