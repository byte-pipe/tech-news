# Bose open-sources its SoundTouch home theater smart speakers ahead of end-of-life - Ars Technica

**SoundTouch API Now Open-Source**
=====================================

Bose has made the application programming interface (API) for its popular SoundTouch speakers publicly available, a move that addresses concerns about the impending end-of-life of expensive home theater devices.

The decision to open-source the API was announced in October, when Bose revealed that its Wi-Fi sound speakers and soundbars would go out of business on February 18. Instead, they will become "dumb speakers," allowing users to control them without relying on internet connectivity.

**Key Features and Changes**
-----------------------------

* The soundTouch speakers will continue to receive security updates until April 6, 2026.
* Without the SoundTouch app installed, users will no longer be able to:
	+ Connect devices via AUX or HDMI.
	+ Use SoundTouch's music integration features like Spotify Connect or AirPlay.
	+ Edit saved presets.
* However, soundTouch device owners who wish to save presets can still use their existing favorites in other music service apps.

**New Feature: Android Auto**
---------------------------

Bose has also provided a workaround allowing users with an Android device to play the same audio simultaneously using SoundTouch devices that support AirPlay 2. The updated app will automatically apply the update on May 6, 2026, without requiring customers to perform any action.

**Open-Source API and Firmware**
-------------------------------

The new API documentation provides independent developers with access to soundTouch's firmware, enabling them to create their own optimized systems for users who prefer such capabilities. This move is seen as a more merciful death, allowing SoundTouch owners who may not want to update or modify their device to do so without restrictions.

**Conclusion**
----------

By making its API and firmware available open-source, Bose has addressed some of the concerns surrounding its devices' impending end-of-life. This decision demonstrates the company's dedication to preserving its products for as long as possible while also preparing users for the transition to more cost-effective alternatives.
