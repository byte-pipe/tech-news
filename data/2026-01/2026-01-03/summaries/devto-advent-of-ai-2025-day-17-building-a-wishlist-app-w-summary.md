# Advent of AI 2025 - Day 17: Building a Wishlist App with Goose and MCP-UI - DEV Community

Here is a concise and informative summary of the article:

**Advent of AI Day 17: Wishlist MCP App**

This post introduces the Winter Wishbox, a digital version of the traditional Winter Fairy's Wishbox that uses a local AI agent called Goose.

* **Key Points:**
	+ Goose is an open source AI agent that extends its capabilities beyond code suggestions.
	+ It can automate complex engineering tasks across startups and existing projects.
	+ Ideal for developers who want to move faster and focus on innovation.
	+ Offers multiple configuration options, seamless integration with MCP servers, and desktop app/C CLI availability.
* **What the Author Built:**
	+ A Wishbox MCP server that renders a visual UI directly in Goose using MCP-UI.
	+ Users can add wishes, view them, grant them when they come true, and delete them as needed.

**About the Project**

A wishlist MCP app for Advent of AI 2025 - Day 17 was built to showcase best practices for integrating AI with various environments like MCP servers.
