# Introducing Cowork | Claude

## Cowork Overview
Cowork: Claude is a simpler way for anyone not just developers to work with Claude in the same way. It's available as a research preview for Claude Max subscribers on our macOS app.

### Key Features
- Giving Claude access to a folder of your choosing, where it can read, edit, or create files.
- Completing tasks like reorganizing downloads, creating spreadsheets, and writing reports with much more agency than in regular conversations.
- Seamless integration with existing connectors for external information.
- Improving its ability to create documents, presentations, and other files using your existing connectors.

### How it Works
- You can set tasks for Claude, which will make a plan and continue working on them until completed.
- Claude is designed to work parallel with you, allowing you to queue up tasks without delay.
- It feels less like back-and-forth and more like leaving messages for a coworker.

## Control Over Claude's Work
- You can choose which folders and connectors Claude can access, ensuring it has explicit permission to do so.
- Claude will ask before taking any significant actions, giving you the opportunity to steer or correct its behavior as needed.
- However, be aware that Caoine may take potentially destructive actions with Claude if instructed, requiring clear guidance on sensitive topics.

## Benefits of Using Cowork
- Simplifies using Claude for non-coding tasks by removing the need to provide explicit context and converting output formats.
- Allows for queueing up tasks and leaving messages in parallel with Claude's work.
