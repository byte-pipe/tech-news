# Kestra.io As Developer Advocate

## Who Am I?

I'm a developer advocate for Kestra.io, joining the company as part of their onboarding and learning journey. I'll be helping expand into the US market.

## What is Kestra.io?

Kestra.io is a declarative workflow orchestration platform that combines everything-as-code and config-only approaches. It provides a good experience for no-coders through its YAML editor, diagram visualization, and editable topology in side-by-side panels.

## Using Kestra

I've recently moved to the Kestra.io team after enjoying walks between coding and non-coding experiences. The platform offers great flexibility in programming choices, making it easier to write code while also utilizing visual tools for non-programs (e.g., modifying YAML directly).

**OSS Edition**

For the general use case, such as single user projects or small-scale organizations, Kestra OSS is a great choice.

**Enterprise Edition**

For businesses seeking custom enterprise solutions with extra features like cloud implementations, leveraging Enterprise features without digging into deeper details requires exploring the respective options for each model (OSS and Cloud).

## Recommended Setup

1. Start by setting up Docker compose within your terminal.
2. Run `docker run --pull=always --rm -it -p 8080:8080 --user=root -v /var/run/docker.sock:/var/run/docker.sock -v /tmp:/tmp kestra/kestra:latest server local`
3. Be cautious when using a persistent database backend.
