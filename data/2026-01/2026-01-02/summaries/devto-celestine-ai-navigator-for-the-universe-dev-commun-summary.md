# Celestine: AI Navigator for the Universe 🪐 - DEV Community

**Celestine: AI Navigator for the Universe**

**Summary**

Celestine is an intelligent, multi-modal AI navigator designed to explore and interact with planets in 3D. It extends Google Maps' intuitive experience to the cosmos by connecting users with their home planet through a terrestrial "twin" found on Earth.

**Key Features**

* Multi-modal interaction: talks or video avatars
* Back to Earth feature visualizes earthly analogues on a 3D globe
* Planetary navigation in 3D interface

**Tech Highlights**

* Frontend: React + react-three-fiber for immersive 3D solar system
* AI Engine: Google's Agent Development Kit (ADK) and Gemini 2.5 with the `Agent Development Kit` framework
* Integration: Analogues Specialist Agent analyzes celestial features and queries the Google Maps Places API to find corresponding Earth analogues

**Technical Requirements**

* Backend: Python (FastAPI) + Docker for containerization on Google Cloud Run
* Frontend: React + react-three-fiber
* AI Engine: Gemini 2.5 with ADK framework

**The Story Behind Celestine**

A project to revive the dream of a "Space Mode" by making cosmic exploration deeply personal and interactive, utilizing Google Maps' navigation tools and AI reasoning about geology.

**Recognition**
