## Examples of Using the Cookbook
---------------------------------

#### Get Current User's Token

Create a token using the `curl` command:
```bash
curl -X POST \
  https://opendata.pro.openai.com/token \
  -u username:openapi \
  -d "grant_type=client_credentials"
```
