## OpenBB-finance - Financial Data Platform for Analysts, Quants and AI Agents


### Overview
OpenBB-finance is a public financial data platform designed for analysts, quants, and artificial intelligence (AI) agents. It provides access to historical financial market data that can be used for research, training machine learning models, or other applications.


* **Key Features**:
  * Historical financial market data with customizable start dates.
  * Support for various time intervals, including daily, monthly, quarterly.
  * Ability to create custom visualizations and charts using openBB's API.
