# How I Use AI Agents + MCP to Fully Automate My Website’s Content - DEV Community

## Automating Podcast Metadata

Adding podcast metadata can be tedious and time-consuming, especially when dealing with multiple episodes across different platforms. To streamline this process, consider using AI agents like Copilot and Playwright MCP along with other tools.

### Before Using AI Agents

*   Manual process of updating website content with new podcast information (e.g., adding a new episode to the site)
    *   Requiring manual copying and pasting from hosting platform
    *   Time-consuming due to repeated steps across multiple platforms
*   Image management using Cloudinary, requiring tedious images downloading and uploading

### Utilizing AI Agents along with Playwright MCP

*   **Playwright MCP**: Extends the capabilities of Playwright, enabling it to navigate to the URL of a hosted podcast and extract relevant metadata.

**Implementing AI Agent with Goose**

*   **Goose**: A coding agent from Block that allows users to create reusable prompts for copying and pasting code blocks.
*   **Integration with Playwright MCP and GitHub MCP**

    *   Using Goose, you can leverage its capabilities in a more flexible way by creating tasks as code using its syntax and structure.

### Example Use Cases

1.  Script a regular expression to automatically extract podcast metadata when adding content.
2.  Create reusable scripts for updating website content with new podcast information, making subsequent updates faster and more efficient.
3.  Utilize automation tools to streamline the image management process using Cloudinary MCP or other similar services.

By leveraging AI agents like Copilot and Playwright MCP alongside Goose, you can automate podcast metadata management, reduce manual effort, and improve overall website maintenance efficiency. The key is to find the right tool for each task, understanding its capabilities, and integrating them effectively to create a seamless workflow.
