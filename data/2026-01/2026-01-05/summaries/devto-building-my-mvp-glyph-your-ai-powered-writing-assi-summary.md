# Building my MVP: Glyph — Your AI-Powered Writing Assistant - DEV Community

**DEV's Worldwide Show and Tell Challenge Submission**

I, Glyph, built an MVP word editor with an AI assistant that helps users write faster, less stressful, and more focused on ideas. Here's a summary of my process:

### Key Points

* Created the world's first AI-powered writing assistant, available for free
* Features include:
 + Writing tool with native, smooth feel
 + AI engine using Llama 3 model (cost-efficient for basic text operations)
 + Video Intelligence feature allows direct upload and transcription support

### The Story Behind It

As a student and developer, I've realized several challenges while writing: inefficiencies in typing, excessive researching tabs, manual transcription of meeting notes. I set out to create an accessible solution for these issues:

* Built full-stack web application with vanilla HTML/CSS/JavaScript (TipTapas core editor)
* Utilized Django API logic, user session management, and other services
* Employed the Groq API with Llama 3 model as the initial AI engine

### Technical Highlights

Glyph is designed for clarity and speed:

* Responsive frontend built with vanilla tech
* Video Intelligence feature available on demand
* Optimized for cost efficiency by using a lightweight LLM for basic tasks
