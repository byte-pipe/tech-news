# Five Projects, Carefully Unhinged and Fully Deployed - DEV Community

**Speedrunning Side Projects as a Matter of Pride**

* **There'sDigital Guilt Trap**: A minimalist experiment in user interaction that applies light emotional pressure via buttons
* **The Checker**: Attempts to analyze personality traits and politely passes judgment, but has also been found emotionally questionable
* **Quantum Weather**: Forecasts are technically accurate but emotionally uncertain
* **A Idle Website**: Does almost nothing yet still manages to require mobile bug fixes due to user needs
* **Personal Portfolio**: Initially built for deployment and exposure of production environments' realities rather than perfection

**Lessons Learned**

* User experience is deceptive
* Mobile layouts can be unforgiving
* Committing code is more productive than overthinking it

**Improvement Ideas**

* **The Checker's**: Consider implementing additional features to address emotionally questionable aspects or refine its judgment-making process
