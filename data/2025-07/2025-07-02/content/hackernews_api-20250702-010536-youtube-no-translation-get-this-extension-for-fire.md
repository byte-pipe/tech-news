This is an open source Add-on preventing automatic translation on YouTube. It ensures that video titles and description remain in their original language and that the audio track defaults to the original version, providing an authentic viewing experience.
FEATURES
 ✨ :
VIDEO TITLES
 : Keep titles in their original language
AUDIO TRACKS
 : Always use the original audio track (also works on shorts)
DESCRIPTIONS
 : Prevent description translations
SUBTITLES
 : Choose your preferred subtitle language. If unavailable, subtitles are automatically disabled (auto-generated ones are always ignored)
