# The electromechanical angle computer inside the B-52 bomber's star tracker

**The Electromechanical Angle Computer Inside the B-52 Bomber's Star Tracker**

### Historical Context

* Before GPS, navigating aircraft relied on celestial navigation techniques using stars, planets, or sun positions.
* Celestial navigation is accurate but requires manual effort and cannot be jammed; it does not require broadcast infrastructure.

### Development of Automated Systems

* In the 1960s, an automated system for tracking stars was developed in the B-52 bomber to perform star navigation manually with complex electromechanical analog computers like the Angle Computer.
* The Angle Computer is an electromechanical device that performs trigonometric calculations using an "electromechanical analog computer" to track and compute angles.

### Operation of the Angle Computer

* The Angle Computer is equipped with a "celestial sphere" mechanism inside, which moves a pointer representing star position.
* The corresponding angles (azimuth and altitude) are read out through electronic devices called synchros, providing information to the navigation system via bundled wires.
* The Astro Compass is one piece of the overall Astro Compass system, consisting of multiple components including an "astro tracker," optical system, and mechanical systems.

### Key Components

* **Angle Computer**: Complex electromechanical device with a computer-like brain that performs calculations using celestial bodies' positions.

* **Gyroscopic Platform**: Maintains telescope position even as the aircraft tilts and moves.
