# Announcing AnalogJS 2.0 ⚡️ - DEV Community

Here is a concise and informative summary of the AnalogJS 2.0 release, while maintaining clarity, coherence, and the original meaning:

**Announcement**

We are excited to announce the 2.0 release of AnalogJS, providing developers with faster support for building websites and applications using Angular.

**Key Features**

The new version includes several key features that enhance development speed, such as:

* First-class support for the Vite ecosystem (e.g., Vitest, Playwright) and Nitro
* Filesystem-based routing and more robust server and deployment capabilities

**What's New**

Analog 2.0 introduces significant enhancements to content resources, including:
* Content files used to display lists of loaded content
* Smaller install and bundle size optimizations
* Vite ecosystem upgrades and new features (e.g., PostsComponent)

**New Features**

Some notable new features included in Analog 2.0 include:

* Hybrid SSR/SSG support with sitemap and RSS feed integration
* Angular CLI/Nx workspace integration (with server and deployment support)
* Smaller content file resource resources to improve load times

Overall, the AnalogJS 2.0 release focuses on speeding up development time while maintaining a robust foundation for Angular applications.
