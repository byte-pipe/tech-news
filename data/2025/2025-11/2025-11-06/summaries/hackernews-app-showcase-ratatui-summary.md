# App Showcase | Ratatui

*   Aatuin: An app that replaces the existing shell history with a SQLite database, records additional context for commands.
*   Bandwhich: A CLI utility for displaying current network utilization by process, connection, and remote IP/hostname.
*  Binsider: A tool for performing binary analysis in your terminal.
*   Bottom: A customizable cross-platform graphical process/system monitor for the terminal.
*   Crossword: A game that plays crossword puzzles in the terminal.
*  /csvlens: A command-line CSV file viewer, similar to `less`.
*   Dua: A disk space analysis tool for speed and rapid deletion of unnecessary data.
*   Fzf-make: A command-line tool that executes make targets using a fuzzy finder with preview windows.
*   gitui: A TUI (User Interface Tool) for Git written in Rust.
*   Gpg-tui: A Terminal User Interface for GnuPG.
*   Joshuto: A terminal file manager written in Rust, similar to ranger.
*   Material: A material design color palette for the terminal.
*   Minesweep-rs: A mine sweeping game written in Rust.
