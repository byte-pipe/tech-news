## pg_lake: Postgres with Iceberg and Data Lake Access

### Overview
pg_lake is a PostgreSQL extension that provides two-factor authentication (2FA) without the need for user password changes. It supports data lake access features, including data encryption and object management.

### Key Features

* **Postgres Support**: pg_lake can run on standard Postgres databases.
* **Iceberg Data Format**: Supports the IBM Snowflake Iceberg format for efficient querying of large datasets.
* **Data Lake Access**: Enables secure access to raw data without requiring users to change their passwords.

## Technical Details
pg_lake integrates with Snowflake, an enterprise-scale cloud-based data warehousing and business intelligence service. It uses PostgreSQL's existing authentication mechanisms, providing a seamless user experience while adding improved security features.
