# Ghc now runs in your browser - Show and Tell - Haskell Community

#### GHC in Browser Summary

*   **Introducing GHC in Browser**: The Ghc team has announced a new way to run Haskell code in web browsers.
*   **Benefits**: This approach allows users to execute Haskell code on the fly without downloading or installing any software.
*   **Architecture**: The program is based on the Haskell Web App, which is built using the WebSockets technology and allows for real-time communication between the client and server.
*   **Server-Side Haskell**: Haskell is used as a functional programming language for the server-side, enabling complex computations and data management tasks.
*   **Client-Side RISC Assembly**: The client is built using RiscOS assembly code, which provides low-level memory and CPU instruction control.
*   **Testing Frameworks**: The project includes several testing frameworks to verify the correctness of the implementation.
*   **Future Development**: The team aims to extend this technology for building larger projects and more sophisticated applications.

The Ghc in browser approach offers various benefits, including:
