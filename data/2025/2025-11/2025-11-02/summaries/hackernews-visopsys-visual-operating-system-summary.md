# Visopsys | Visual Operating System

**Visopsys Overview**

Visopsys is an alternative operating system designed for PCs since 1997. It targets small and fast systems, providing a simple but functional graphical interface with pre-emptive multitasking capabilities.

**Key Features:**

* Small and fast operations
* Graphical user interface (GUI)
* Fully preemptive multitasking support
* Protected mode to ensure security
* Open source and free software

**History of Visopsys**

The operating system has been in development since 1997, with subsequent versions being released as follows:

* Version 0.92: June 2023 (unofficial update)
* Version 0.91: July 2021
* Version 0.9: April 2020
* Version 0.85: May 2019
* Version 0.84: August 2018
* Older versions: various release dates throughout the year

Note that only unofficial updates are provided in this summary, to reflect the OS's ongoing development and limited availability.
