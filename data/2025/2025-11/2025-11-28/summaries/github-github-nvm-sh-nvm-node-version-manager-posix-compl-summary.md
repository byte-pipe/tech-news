# GitHub - nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

## nvm-sh - Node Version Manager with POSIX-compliant Bash Script

### Overview
nvm-sh is a tool that manages multiple active Node.js versions on Windows.

### Key Features:

* Manage multiple Node.js versions
* POSIX-compliant bash script to handle different environments
* Active node version control

### License
MIT license

### Usage
* To use nvm-sh, navigate to the installation directory and run `nvm init --user`

## Installation and Configuration
1. Install Node.js on your system if you haven't already.
2. Navigate to the installed Node.js path (e.g., `C:\Program Files\nodejs`)
3. Run `nvm install` to enable nvm-sh
4. Use `nvm use <version>` or `nvm reload` to switch between different versions
