# Molly

### Molly

Molly is an independent Signal app for Android with the following improved features:
- **Fully Open Source (FOSS)**: No proprietary blobs, unlike traditional Signal apps.
- **End-to-end encrypted**: Database protection using passphrase encryption for secure communication.
- **Multi-device support**: Sync data across multiple devices in a single account.
- **Material You theme**: Customizable color scheme based on device preferences.
- **UnifiedPush**: Efficient notification system that doesn't rely on third-party services.
- **Automatic Locking**: Temporary removal of app access when absent for set periods.
- **RAM shredding**: Secure deletion of sensitive data from storage.

### Features
* Supports [torbrowser](https://github.com/lorien/orbot): Includes Tor integration and SOCKS proxy functionality.
* Offers additional new features planned to be released.
