# Announcing AnalogJS 2.0 ⚡️ - DEV Community

## AnalogJS 2.0 Release Announcement

The Angular team is excited to announce the release of version 2.0 of AnalogJS, a powerful and feature-rich meta-framework built on top of Angular.

### Key Features in Version 2.0

*   Enhancements to the Vite ecosystem with improved support for new tools and services
*   New features such as content resources for displaying lists and files, smaller install and bundle size optimizations, and upgrades to the Vite ecosystem
*   Smaller bundle sizes mean faster load times and improved user experience
*   Support for hybrid Server-Side Rendering (SSR) and Static Site Generation (SSG)

### New Features

#### Content Resources

*   Now includes resources for displaying content lists and files
*   Allows loading of pre-rendered content and reducing the amount of data being transferred to the client side

#### Enhanced Bundling

*   Improved support for smaller bundle sizes, resulting in faster load times

## What's Changed

### Routes and Navigation

*   Better integration with Angular's latest navigation features and routing system
*   Support for server-side rendering and static site generation

### Server-Side Rendering (SSR) and Static Site Generation (SSG)

*   Hybrid SSR/SSG support enables the creation of hybrid applications that seamlessly transition between server-side and client-side rendered content.

### Bundling Improvements

*   Smaller bundle sizes result in faster load times and improved user experience.
