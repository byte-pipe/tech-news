**nvm-sh: A Node Version Manager for POSIX-compliant Bash Scripts**
=================================================================

### Overview

The `nvm-sh` package provides a POSIX-compliant bash script for managing multiple active node.js versions. It offers features such as automatic branch management, tracking commits, and notification settings.

### Key Features

*   POSIX-compliant bash scripting
*   Automatic branch management
*   Tracking of commits
*   Notification settings customizable with signing in
