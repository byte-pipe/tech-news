# PaddlePaddle/PaddleOCR
---------------------

### Overview

* A lightweight OCR toolkit that bridges images and PDFs with LLMs (Large Language Models), supporting 100+ languages.
* Open-sourced, actively maintained, and widely used for automated document processing tasks.

### Key Features

* Automatic text detection and recognition
* Supports various image formats, including PDFs
* Integrates with Large Language Models (LLMs) for precision OCR
* Optimized for performance, making it suitable for resource-constrained environments.
