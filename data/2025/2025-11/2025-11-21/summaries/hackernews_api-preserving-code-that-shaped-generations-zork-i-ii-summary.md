# Preserving code that shaped generations: Zork I, II, and III go Open Source

### Preserving Code That Shaped Generations: Zork I, II, and III Go Open Source

**Introduction**

Microsoft's Open Source Programs Office (OSPO), Team Xbox, and Activision are teaming up to preserve historic games like Zork I-III. This collaboration aims to make these groundbreaking games accessible to students, teachers, and developers for educational purposes.

**The Games' Revolutionary Engine: The Z-Machine**

Zork I-Ill, II, and III were games developed by Infocom, a renowned game development company that designed the immersive world of words and imagination. Despite being released in 1979-1981, these games predated graphical gaming by over three decades.

### Zork's Engineering Legacy

*   The **Z-Machine**, built into these games, is an original source code specification for a virtual machine, ensuring seamless execution on various platforms.
*   This unique approach allowed Infocom to create cross-platform games by using the Z-Machine as a base system.

**Preserving History**

Game preservation involves multiple aspects: research and play. The **Zork** source code will be preserved through upstream pull requests submitted to historical source repositories, adding the game's open-source grant.
*   Each repository includes:
    *   **Source Code**: The source code for Zork I, II, and III, including build notes, comments, and relevant files.
    *   **Documentation**: Accompanying documentation provides insights into historical context and functionality.
