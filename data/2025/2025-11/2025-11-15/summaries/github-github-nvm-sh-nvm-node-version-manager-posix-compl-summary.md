# GitHub - nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

* The `nvm-sh` package is a Public Node Version Manager developed for POSIX-compliant bash scripts.
* It provides features such as managing multiple active node.js versions, displaying version information, and enabling signing into notification options.
* Signing in to change notification settings requires the user to authenticate via GitHub authentication.
