# GitHub - nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

## Node Version Manager - POSIX-compliant Bash Script

### Overview
This is a bash script for managing multiple active Node.js versions. It provides the functionality to switch between different Node.js versions.

### Key Features

* Requires Node.js and npm v8
* Offers multiple versions of Node.js supported (9.6k+)
* Includes an interface for switching between different Node.js versions

### Summary

This script is designed for managing multiple active Node.js versions using the POSIX-compliant bash interface. It supports two main options:

- **Switching**: Navigates through all installed Node.js versions to select and execute a specific one.
- **Managing versions**: Displays available Node.js versions, their respective descriptions, and helps in making version changes.
