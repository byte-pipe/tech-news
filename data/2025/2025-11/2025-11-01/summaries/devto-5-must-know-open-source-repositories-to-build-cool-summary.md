# 5 must know open-source repositories to build cool AI apps - DEV Community

## Building Real-Time AI Apps using Open-Source Frameworks

With the momentum of AI-powered features gaining traction across teams, developers must be able to build impressive applications without relying on expensive budgets. The key to successful development lies in choosing the right open-source tools and frameworks that provide full control, transparency, and freedom for experimentation.

### Tool Overview 1: Stream Vision Agents

Stream Vision Agents is an open-source framework for building real-time, multimodal AI that brings true intelligence to live video. It's designed for developers who want to bring intelligent capabilities to live video without being locked into a single model or transport provider.

* Key Features:
	+ Open Source Fork and Improvements
	+ Works with Stream Video SDK or WebRTC-based SDKs
	+ Flexible Provider Plug-ins (OpenAI Realtime, Gemini Live, etc.)
	+ Real-time Vision and Multimodal Intelligence

### Tool Overview 2: Open-Sora

Open-Sora is an open-source framework that converts text or images into short, high-quality videos. It provides a valuable tool for developers who need to fine-tune their own datasets or generate domain-specific content like marketing clips or story scenes.

* Key Features:
	+ Supports Text-to-Video and Image-to-Video Generation
	+ Fine-Tunning on Datasets (Domain-Specific Content)
	+ Initial Early Release with Room for Experimentation
