# Windows 11 to add an AI agent that runs in background with access to personal folders, warns of security risk

# Windows 11 Experimental Feature: Agent Workspace

Windows is introducing a new experimental feature called "Agent Workspace" for its upcoming operating system, Windows 11. This is part of a broader push towards turning Windows 11 into an AI-native OS amidst "Copilot" backlash from users who don't want AI integration in their devices.

## Key Features and Benefits

1. **Experimental agentic features**: Agent Workspace will allow AI agents to access the most-used folders on your desktop, like Desktop, Music, Pictures, and Videos.
2. **Runtime and desktop for AI agents**: This feature enables AI agents to have their own runtime, desktop, user account, and ability to run in the background whenever prompted.
3. **Background execution**: If enabled, Agent Workspace will keep running continuously without interruption.

## Introduction to AI Agents

AI agents are programmed to perform tasks on your behalf, often mimicking human behavior. They can browse through websites, research topics, and even conduct tasks like booking trips or editing files.

## Key Differences from Existing Features

* **More control over the Agent workspace**: Each agent has its own isolated workspace with limited access controls.
* **No need to switch accounts**: You remain in control of your normal desktop while using AI agents' sessions.

The feature is currently only available to Windows Insiders in the Dev or Beta Channel and will require testing outside of this group for further refinement.

## Impact on Security

Microsoft has cautioned about potential security risks associated with Agent Workspace, but it's also possible that more advanced measures can mitigate these concerns.

### Expected Improvements Over Existing Features

By integrating AI into its software, Microsoft aims to enhance user experience, enable new possibilities for productivity and exploration, while addressing the need for secure access controls and mitigating risks.
