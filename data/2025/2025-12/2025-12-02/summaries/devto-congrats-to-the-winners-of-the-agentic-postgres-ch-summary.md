# Congrats to the Winners of the Agentic Postgres Challenge with Tiger Data! - DEV Community

**Agentic Postgres Challenge with Tiger Data Winners**

* **Successful Competition**: The Agentic Postgres Challenge with Tiger Data successfully collected innovative and creative solutions from various participants.
* **Challenge Highlights**:
    * Participants can experiment with AI on Postgres, building something that was considered "outside the box".
    * Examples include a multi-agent code review system, real-time fraud detection system, AI-powered platform for GitHub repositories, and more.
* **Judging Challenges**: Judging became significantly harder due to the creativity, technical depth, and experimentation shown by participants.
* **Winning Solutions**:
  * Simran Shaikh's (```@simranshaikh20_50built```): A multi-agent code review system that achieved 4x faster performance using zero-copy database forks from Tiger Cloud.
  * Mayuresh (```@mayu2008created```): FraudSwarn, a real-time fraud detection system that combines pg_text and pgvector for improved accuracy.
* **Technical Details**:
    * Simran's system uses parallel analysis within isolated database forks.
    * Mayuresh's system innovates with hybrid search using pg_text and pgvector.
