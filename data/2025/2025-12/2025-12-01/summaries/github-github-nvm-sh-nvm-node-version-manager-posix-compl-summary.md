# GitHub - nvm-sh/nvm: Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

**Node Version Manager - POSIX-compliant bash script**
====================================================

### Overview

The article discusses the `nvm-sh`/`nvm` package, a Node.js version manager that provides a POSIX-compliant bash script for managing multiple active node.js versions.

### Key Points

* The `nvm.sh` file is used to manage Node.js versions and create an environment with specific settings.
* Installation: `curl -o nvm.sh https://raw.githubusercontent.com/nvm-sh/nvm/master/shim.nvm && source nvm.sh`
* Signature required: "nvm <version>"
* Version manager automatically updates: "nvm install"
* Can switch between different Node.js versions
* Supports multiple environments (e.g., development, testing)
