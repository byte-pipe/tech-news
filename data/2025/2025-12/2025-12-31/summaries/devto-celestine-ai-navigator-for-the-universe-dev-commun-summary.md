# Celestine: AI Navigator for the Universe 🪐 - DEV Community

**Celestine: Intelligent AI Navigator for the Universe**

This is an innovative project aimed at creating an immersive exploration experience that combines Google Maps with artificial intelligence. The goal of Celestine is to allow users to explore planets in 3D, interact with their environment, and discover secrets while seamlessly navigating between celestial bodies.

**Key Features and Highlights**

* **Multi-modal interaction**: Users can engage with Celestine via voice or a real-time generated video avatar powered by Tavus.
* **Planetary navigation**: The application provides seamless travel between celestial bodies using 3D geometry.
* **Back-to-Earth functionality**: Visualize the coordinates of earthly analogues on a 3D globe.
* **Intelligent AI co-pilot**: Utilizes Google Maps Platform's Places API to reason about geology and perform semantic searches across planets.

**Technical Details**

* **Frontend**: React + react-three-fiber for immersive 3D solar system rendering
* **AI Engine**: Built with Google's Agent Development Kit (ADK) and Gemini 2.5
* **Backend**: Python (FastAPI), containerized with Docker, and deployed on Google Cloud Run
* **Specialist Agent**: The "Analogues Specialist" agent analyzes celestial features to autonomously query the Places API

**Aims and Inspiration**

This project aims to revive the dream of a "Space Mode" while creating an interactive and personal experience. It seeks to integrate AI technology into the existing Google Maps platform, making it accessible for both enthusiasts and professionals.

**Recognition History**

Celestine was originally built as part of the Google Maps Platform Hackathon, where it received recognition as a nominee.

The project is now showcasing its technical capabilities on the Google Maps Platform website.
