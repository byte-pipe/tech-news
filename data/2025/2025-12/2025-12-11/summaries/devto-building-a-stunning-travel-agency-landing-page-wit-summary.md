# Building a Stunning Travel Agency Landing Page with Uno Platform — My WOW Factor Challenge Submission 🤩 - DEV Community

**Building a Stunning Travel Agency Landing Page with Uno Platform - My WOW Factor Challenge Submission**

**Submissions Overview**
This is a submission for the AI Challenge for Cross-Platform Apps - WOW Factor on the Uno Platform Hot Design Challenge.

**Project Background**
After completing my first project for the **Uno Platform Hot Design Challenge**, I thought the journey was over. However, something unexpected happened. I found myself enjoying the process so much that I couldn't stop. This creative flow and instant feedback from across platforms pulled me right back in.

**Second Prompt - WOW Factor**
When I went back to the challenge and read more about **WOW Factor**, I felt like an invitation I couldn't ignore. This time, instead of a mobile UI, I decided to build something bigger and more immersive: A modern, clean, travel-agency landing page entirely powered by Uno Platform.

**Rationale**
The primary goal was to create a visually engaging home screen that would grab the users' attention with creative visuals, mock data, animations, and smooth performance. My goals included:

* Creating a welcome section that feels alive
* Smooth vertical scrolling
* Destination cards using mock data
* Subtle animations for delight without slowing down performance

**Design Vision**
My design vision consisted of:

* A hero section that feels alive with large images, layered layout, and a welcoming confetti animation after subscription.
* Smooth vertical scrolling
* Destination cards with clean and rounded corners, subtle shadows, and responsive images
* Subtle animations for effect without slowing down performance

**Implementation Using Uno Platform**
The project was built using mock data and visual assets. I used Grid & StackPanel layout composition and a lightweight confetti animation to create the desired impact.

## Challenges Faced
Cross-platform UI design doesn't need to be painful or slow, as Uno Platform helps bridge this gap.

### Challenges:
1. **Cross-Platform Performance**
  - Created animations with minimal impact on performance.
2. **Custom Visuals**
  - Designed custom visuals using mock data that blend seamlessly into the platform.
3. **Clean Design**
  - Implemented a clean, structured XAML design that is easily extensible.

**Results**
The WOW Factor prompt asked for:

* A visually engaging home screen
* Fun theme (coffee shop style)
* Creative visuals, mock data, animations
* Smooth performance
* Something that instantly grabs attention

I successfully met these requirements and achieved a modern, clean travel-agency landing page on the Uno Platform.

**Conclusion**
This project showcased my capabilities in designing a stunning cross-platform landing page using Uno Platform. By understanding the challenges and applying the right techniques, I was able to create an immersive experience that exceeded expectations.
