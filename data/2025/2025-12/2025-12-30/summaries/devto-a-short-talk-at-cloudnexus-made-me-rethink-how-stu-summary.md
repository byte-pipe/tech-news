# A short talk at CloudNexus made me rethink how students are using AI - DEV Community

**Agentic AI and Student Use of AI Tools**

A recent talk at the AWS Cloud Club QAU event focused on cloud technologies and Agentic AI, a concept where students are using AI tools to augment their learning. The speaker was surprised by how confident students claimed they were already "using AI correctly" without understanding its limitations.

**Key Findings**

* Students were primarily using AI tools for code generation (ChatGPT) rather than deep understanding or real-world application.
* However, this approach did not hinder debugging or project development.
* Instead, AI tools like ChatGPT are more effective when used for specific tasks like:
	+ Understanding concepts
	+ Breaking down ideas
	+ Writing content
	+ Asking "why" questions

**The Importance of Context**

Not all AI tools were created equal. Simple chat-based tools, like those provided by CloudNexus, shine best in certain contexts (e.g., development work) and should not be relied upon as primary coding engines.

**Amazon Q: A Development-Focused Tool**

AWS Q stood out because it provides cloud-aware guidance, best practices, and security-conscious suggestions. This tool is particularly valuable for developers working within AWS environments.

**Why Cloud AI Matters**

Agentic AI requires a different approach from traditional chat-based tools. It demands infrastructure, scalability, and security, which platforms like AWS Bedrockstart can help provide.

**Conclusion and Takeaways**

Students should focus on using AI tools to augment their learning (Code-focused tools excel in development work). They should understand the limitations of AI in isolation and leverage it for specific tasks only. Ultimately, students need to develop the ability to "think for themselves," rather than relying solely on Agentic AI tools.

**Summary Key Points:**

* Students use ChatGPT mainly for code generation.
* Agentic AI is not about prompts; instead, it's about delegation with context and goals.
* Cloud-based tools like AWS Q are essential for development work.
* Students should focus on using AI to augment their learning, rather than relying solely on Agentic AI tools.
