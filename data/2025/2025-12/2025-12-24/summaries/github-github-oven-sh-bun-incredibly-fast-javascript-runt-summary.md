# GitHub - oven-sh/bun: Incredibly fast JavaScript runtime, bundler, test runner, and package manager – all in one

**oven-bun: A Single Frontend Runtime**
=====================================

### Overview
------------

**Bun** is a fast JavaScript runtime that serves as a bundler, test runner, and package manager for the frontend. It includes all three features in one application.

### Key Features
----------------

*   **Bundling**: Quickly bundles dependencies to create and serve executable code.
*   **Test Runner**: Allows developers to run tests with a single command.
*   **Package Manager**: Manages dependencies and packages required by the project.

### Usage and Contributions
-----------------------------

To use `bun`, follow these steps:

1.  Fork the repository and commit changes.
2.  Install `bun` using yarn or npm.
3.  Run `bun build` to create a production-ready bundle.
4.  Set up test coverage by running `bun test`
5.   You can configure notification settings by editing your environment variables.
