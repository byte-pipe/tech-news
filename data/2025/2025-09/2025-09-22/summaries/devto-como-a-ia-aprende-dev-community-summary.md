**Perguntas Fundamentalas**

Uma pergunta fundamental é: Como a IA "aprende" como você ouviu?

A resposta parece clara: não porque ela aprende diretamente como um ser humano, mas sim como um modelo treinado. Mas esses modelos precisam de dados para serem treinados e, consequentemente, aprender.

**Indicadores do Mercato**

Para entender quais problemas a IA está sendo desafiada, é necessário analisar o mercado.

- **GitHub Copilot**: um modelo que usa dados coletados no GitHub para sugerir código.
- **AI-Powered Tools de Trabalho**: como os que usam tokenização e aprendizado por referência.

**Complexidade Técnica**

Como a IA aprende? Existem muitos conceitos que devem ser compreendidos:

- **Modelos de Aprendizagem**: são modelos que aprendem ao se environmentar com um conjunto de datas (dados).
- **Tokens**: pedaços menores de texto, como sílabas ou palavras inteiras, que se combinam para criar uma frase.
- **Aprendizado Supervisionado e Não Supervisionado**: podem ser usados em combinação ao longo do processo.

**Viabilidade Economática**

Para obter esses resultados, é necessário considerar alguns fatores econômicos:

- **Prazos de Trabalho**: para treinar um modelo, pode levar semanas ou até meses dependendo do tipo.
- **Distribuição de Custos**: se houver muitas empresas que precisam aprender a usar o mesmo modelo sem gastar em manutenção.
