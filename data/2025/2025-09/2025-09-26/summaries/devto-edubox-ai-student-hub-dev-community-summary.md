**Introduction**

EduBox is an intelligent student hub designed to help users manage their academic life more effectively.

**Key Features and Benefits**

1. **Smart File Management**: AI-powered file management with semantic search.
2. **Intelligent Planner**: An AI-driven planner that learns user patterns and helps avoid last-minute cram sessions.
3. **Campus Life Hub**: A feature that connects the user to campus clubs and events.
4. **AI Assistant**: A personalized academic organizer that understands the user's studies.
5. **Real-time Syncing**: Real-time sync technology for seamless data exchange.

**Technical Overview**

1. **Frontend**: Built using Next.js, TypeScript, and styled with Tailwind CSS.
2. **Authentication**: Leveraged Clerk for secure user sign-in and profiles.
3. **Sync/Storage**: Utilized Convex for real-time sync and serverless data operations.
4. **AI & Agents**: A multi-agent system comprised of CopilotKit, LLMs hosted via Vertex/Gemini using Tools, and Kendo RAG.

**Deployment**

The application is currently deployed on Vercel, with environment-driven configuration.

**User Experience**

1. **Landing Page**: Features three.js components.
2. **Dashboard & Overview**: Displays academic progress tracking and analytics.
3. **User Profile & Settings**: Allows users to manage their profile and settings.

**Conclusion**

EduBox offers a comprehensive solution for students who want to stay on top of their academic life without burning out.
