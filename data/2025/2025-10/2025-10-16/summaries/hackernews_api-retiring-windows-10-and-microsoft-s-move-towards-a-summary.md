# Retiring Windows 10 and Microsoft's move towards a surveillance state

I'm switching from Windows 10 to Linux due to concerns over data privacy and the potential forMicrosoft to become a "surveillance state." Here are the key points of my decision:

## Main Reasons for Choosing Linux
- Avoid artificial limitations like TPM and Secure Boot, which can lead to unnecessary hardware upgrades.
- Obfuscated setup options that push users towards purchasing new computers with similar add-ins.

## Microsoft's Design Considerations
- The process of setting up OneDrive as a backup creates data concerns by moving customers' data to the cloud without consent.
- Choosing a Microsoft-branded browser is obfuscated, making it difficult for users to select alternatives.
- Integration AI tool Co-pilot can be installed without user consent and may be removed with updates.

## Surveillance State Concerns
- History tracking tool "Recall" will collect snapshots of user experiences in Microsoft's OneDrive cloud, creating a potential surveillance state.

## Recommended Linux Replacements
If you're unhappy with Windows 10, consider the following Linux distributions:
1. Zorin OS: A Windows-like experience on modern hardware.
2. PopOS: Built for gamers, this distribution is suitable for most users.
3. Ubuntu: Provides an all-around desktop experience and requires modern hardware.
4. Elementary OS: Suitable for minimalist users.
5. MX Linux: Available since 2011, it's a robust choice for those with older hardware.

Considering these points, I recommend switching to Linux if you have Windows 10 installed and want to avoid potential surveillance state concerns.
