### Description

Liquibase has migrated to the Functional Source License, which is not an open source license, as Liquibase itselfacknowledges. But this repository continues to misleadingly characterize Liquibase as an open source project, particularly in various places in the fileREADME.md.

### Steps To Reproduce

Viewgithub.com/liquibase/liquibase, or the contents of the README.md file, to find out about the liquibase community project.

### Expected/Desired Behavior

The README.md file and any similar project documentation will no longer misleadingly suggest that liquibase is still an open source project.
