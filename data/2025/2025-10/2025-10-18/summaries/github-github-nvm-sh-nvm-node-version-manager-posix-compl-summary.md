* Manages multiple active node.js versions
* Utilizes a POSIX-compliant Python library (nvm) under the hood

### Benefits

* Allows users to manage and switch between different node.js versions easily
* Enables developers to try out various node.js versions before committing changes to their project

### Usage

1. Clone the repository or clone a fork of nvm.sh
2. Run `nvm-sh` as an interactive shell in your terminal
