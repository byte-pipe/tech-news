# Improvements in Typst 0.14

## Introduction
Typst, an open-source tool developed primarily for academia, has been expanded into various industries and applications. This blog post focuses on the key features of Typst version 0.14: improved accessibility, support for PDF standards, native image formats, character-level justification, enhanced HTML export, and streamlined upgrade process. These enhancements cater to a broader audience, including those with specific needs such as screen readers.

## Accessibility Enhancements
Typst 0.14 now features:

- **Accessibility by Default**: Generates accessible documents without relying on user input.
- PDF Standards Compliance: Supports image formats and character-level justification for improved readability.
- HTML Export Improvements:

  - **PDFs as Native Images**: Allows use of images directly from files, rather than importing them separately.
  - **Character-Level Justification**: Aligns characters within a paragraph for clearer text layout.

## Migrating to Typst 0.14
This release is suitable for users seeking to upgrade their documents using the web app or command line interfaces:

- **Web App Upgrade**: A prompt appears in the user's browser, guiding them through an easy upgrade process.
- **Command Line Interface Upgrade**: Runners are available; installation instructions can be found at the changelog.
