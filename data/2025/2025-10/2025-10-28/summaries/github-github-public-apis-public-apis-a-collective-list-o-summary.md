# GitHub - public-apis/public-apis: A collective list of free APIs

# Public APIs
=================================


## Overview
----------------


The **public-apis** GitHub repository provides a collective list of free publicly available APIs. This list is accessible through the [APILayer API](https://apilayer.io/apiv3).


## Key Features
----------------


*   A curated collection of 374k+ Starred APIs, 39.5k Forks, and various project repositories ( branches, tags).
*   Included activities include watching subscription status errors and a "Forked" counter.

## API Details
---------------


*   The **MIT license** ensures the free usage of this information.
*   This list is designed to be easily accessible for developers and teams looking to expand their API offerings or leveraging open technologies.


## Watch vs. Starred vs. Forked
-------------------------------


The key differences between these indicators are:


*   **Watch**: Displays a button allowing users to watch the subscription status of an API.
*   **Starred**: A badge indicating that an API has been successfully watched from the user's repositories during a single pull request.
*   **Forked**:

In this context, we use:
*   "Branches" as the indicator instead of `forks`
*   "Tags" when discussing features not provided by the repository being forked
*   Use Fork stars in actual cases
