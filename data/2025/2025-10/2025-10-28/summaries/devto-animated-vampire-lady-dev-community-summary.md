# Animated Vampire Lady - DEV Community

**Animated Vampire Lady - CSS Art Submission**

### Inspiration

The artist wanted to create a realistic portrait of a female vampire using CSS. The idea inspired by the "Realistic Portraits with CSS" challenge.

### Demo and Journey

- Initial idea: Self-portrait or vampire lady
- Blocking out areas: Vampire's face, eyes, mouth
- Positioning: `position: absolute;`
- Blood drips and animation for interest
- Blinking: Using keyframes for eyelid size control and creating "blink" effect
- Peeking through castle window using grid and offsetting rows with margin

### Problem Identification

- Bricks were difficult to create from scratch, used grid and offsetting rows with margin
- Adding evil peering eyes in holes required some experimentation
- Removing bricks and placing them back for a natural look needed adjustment

### Solution

- Modifying the blinking effect by having it "organic" rather than synchronized
- Replacing bow in hair with dramatic rose shape
- Adding shading to face using shapes and gradients

### Outcome

- A realistic portfolio that meets the challenge of showcasing CSS art.
