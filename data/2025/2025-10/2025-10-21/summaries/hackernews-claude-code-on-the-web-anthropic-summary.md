# Claude Code on the web \ Anthropic

## Introduction to Claude Code on the Web

Claude Code on the web is a new way to delegate coding tasks directly from your browser. It's available now in research preview for Pro and Max users.

## Key Features

- Runs multiple coding tasks in parallel across different GitHub repositories
- Real-time progress tracking with automatic PR creation and change summaries
- Isolated environment with sandboxing restrictions for secure execution
- Integrates seamlessly with existing Claude Code workflows (as part of an iOS app)
- Customizable network configuration to choose domains for connections

## Benefits

- Faster development and faster feedback loops for bug fixes, routine tasks, and parallel work
- Easy-to-use interface for managing coding sessions on a single screen
- Suitable for productivity platforms that use multiple repositories and projects
- Can be integrated with other cloud-based services for streamlined workflows

## System Requirements

* GitHub account for project management
* Claude Code installation installed (as part of Pro or Max)
* Google Cloud Platform Account to enable secure proxy service
* iOS app for mobile deployment

## Getting Started

Visit the official website ([CLAUDIE.com/codeto](http://claude.com/codeto)) to connect your first repository and start delegating tasks.
