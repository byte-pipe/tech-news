# Congrats to the latest KendoReact Free Components Challenge Winners! - DEV Community

### KendoReact Free Components Challenge Winners Announce

The winners of the recent KendoReact Free Components Challenge have been announced. The competition consisted of various use cases, including an intelligent student hub (EduBox), a game that challenges AI-generated images (Bot or Not), and a revolutionary healthcare application (Consultify).

### Overall Prompt Winner: EduBox

Arya Pratap Singh created the Intelligent Student Hub (EduBox) as the overall prompt winner. The project tackles the problem of managing academic life across multiple disconnected tools, offering features such as semantic search, AI-powered planning, campus life integration, and an AI assistant.

#### EduBox Features:

*   File management
*   Semantic search
*   AI-powered planner
*   Campus life integration
*   AI Assistant

### Code Smarter, Not Harder Winner: Bot or Not

@kaoutar_benhadine_9c91300 built the engaging game (Bot or Not) with features such as Daily Challenge mode, Streak Mode, and a complete admin panel for content management.

#### Bot or Not Features:

*   Game challenges
*   Progress-based difficulty
*   Communication with multiple languages

### RAGs to Riches Winner: Consultify

@iamceejaytackled the healthcare challenge (Consultify), which aims to overcome language barriers that cost lives across Africa by implementing context-aware communication and multilingual tools.

#### Consultify Features:

*   Telemedicine
*   Context-aware RAG (Recognizing Association of Gadgets)
*   Multilingual communication

### Sustainability for Dev Community

The winners were awarded $1,000 USD, an exclusive DEV Badge, and a DEV++ Membership. The community was also encouraged to participate in the next challenge by following relevant tags.

This concludes the announcement of the KendoReact Free Components Challenge winners. If you have any further questions or need assistance with your own projects, feel free to ask!
