# Agent Skills: Customized AI for Task-Driven Workflows

## Introduction

**Agent Skills**

Claude, an artificial intelligence developed by Anthropic, now allows users to customize its performance and tasks through a new feature called "Skill" folders. These folders include instructions, scripts, and resources that Claude can load when needed, improving its ability to perform specific tasks.

## How Agent Skills Work

1. **Accessing Skills**: When a task is relevant to the user's workflow, skills with matching names are detected by Claudes' natural language processing capabilities.
2. **Loading Minimal Information**: Only minimal information and files from relevant skills are loaded, preserving Claude's fast performance while still providing specialized expertise.

## Types of Agent Skills

* **Composable**: Can be combined to create a specific workflow stack
* **Portable**: Use the same format across various applications (Claude apps, Claude Code, API)
* **Efficient**: Only load what's needed and reused
* **Powerful**: Incorporates executable code for complex tasks when traditional programming is not suitable

## Integration with Other Anthropic Products

* Available to all Claude products: Pro, Max, Team, and Enterprise users
* Skills are added through the "Settings" menu for existing Enterprise users
* Enables developers to create custom skills using a tool called Code Execution Toolbeta and the "new/v1/skillsendpoint"

## Benefits for Users

* Improved productivity: Automate repetitive tasks with ease
* Enhanced customization: Tailor performance to specific needs without requiring manual data processing
* Integration with external tools: Interact with professional-grade Excel spreadsheets, PowerPoint presentations, Word documents, and PDFs
