# Microsoft is plugging more holes that let you use Windows 11 without an online account | The Verge

# Microsoft Cracks Down on Creating Local Accounts in Windows 11

Microsoft has taken steps to remove known workarounds for creating a local account on Windows 11, causing issues during setup. In a new test build released today, the company is ending support for bypass methods, including using "bypassnro" and disabling the "start ms-cxh:localonly" command.

## Key Points

* Microsoft is removing known workarounds to create a local account in the Windows Setup experience
* Users will need to complete OOBE screens with an internet connection and Microsoft account for future versions
* This change affects users who have previously used these methods, such as custom user folder names using "start ms-cxh:localonly"
* Alternative methods may be implemented later, including adding a command to name the default user folder during setup (expected to make the process simpler)

## What It Means for Users

Following this change, Windows 11 users will need to:

* Complete OOBE screens with internet access
* Use an authenticating device or Microsoft account to complete the sign-in process in future versions
* Be more cautious when using workarounds, such as custom folder names
