## Node Version Manager - POSIX-compliant Bash Script
=============================================

### Overview
------------

The `nvm-sh` is a Node.js version manager script, allowing multiple active versions of Node.js to be managed. This script provides an easy-to-use interface for switching between different versions.

### Key Features
----------------

*   **POSIX compliance**: The script adheres to the POSIX standard, ensuring compatibility with Linux systems.
*   **Bash-based**: It uses Bash scripting language, providing a familiar and interactive menu-driven experience.
*   **Easy node.js version selection**: The user can switch between different versions using predefined shortcuts.
