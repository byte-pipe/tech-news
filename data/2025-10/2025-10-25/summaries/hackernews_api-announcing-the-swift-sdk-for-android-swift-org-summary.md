# Announcing the Swift SDK for Android | Swift.org

# Introduction to the Swift SDK for Android

The Swift SDK for Android provides developers with a powerful toolset to build native iOS and macOS applications on top of the Android operating system. By leveraging Swift, developers can take advantage of Swift's great interoperability and create apps that seamlessly integrate across platforms.

## Key Features

The Swift SDK for Android includes:

* Interoperability: Share code between Swift and Java libraries
* Native Code Generation: Generate safe and performant bindings to bring business logic from one language to the other
* End-to-End Application Workflows: Demonstrate how to build comprehensive applications on Android using Swift

## Getting Started with the SDK

 Developers can obtain the full set of APIs with nightly preview releases, which are updated regularly. The SDK is also available as a downloadable package that can be used on Linux or macOS.
