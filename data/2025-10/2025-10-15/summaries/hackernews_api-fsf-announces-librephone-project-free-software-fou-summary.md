# FSF Launches Librephone Project for Mobile Phone Freedom

by

Free Software Foundation

Published on

Oct 14, 2025 03:09 PM

BOSTON, MASSACHUSETTS, USA - The Free Software Foundation (FSF) announced its project to bring mobile phone freedom to users today. "Librephone" is an initiative aimed at reversing obstacles preventing mobile phone autonomy until its goal is achieved.

**Background**

For forty years, the FSF has advocated for computing freedom in desktop and server computers. After transitioning to mobile phones, they continue to work on bringing the right to understand, modify, study, and share software used by users daily to mobile phones. "Forty years ago," Kooyman said, "our focus was on providing an operating system people could use on desktop and server computers in freedom."

**Objectives**

Librephone aims to close gaps between existing Android distributions and software freedom. By prioritizing the free software work already done by not entirely free mobile phone operating systems like LineageOS.

**Technical Project Lead**

Rob Savoye, a developer, will lead the technical project. He's researching device firmware and binary blobs in other mobile phone freedom projects to optimize the free software work on Librephone.

**Funding**

Initial funding comes from John Gilmore, FSF board member, who used LineageOS running MicroG and F-Droid before discovering proprietary modules in standard Android phones. This experience led him to collaborate with Savoye to reverse-engineer these non-free modules.

**Steps**

1. **Triaging existing packages**: Identify problematic software on phones.
2. **Compatibility analysis**: Find a non-Free OS phone.
3. **Reversing and replacing non-Free modules**: Replace proprietary firmware components.
4. **Existing development support**: Collaborate with other free software projects that aim to build new free Android-compatible operating systems.

The FSF is committed to bringing freedom and full autonomy to mobile phones, which they believe has been delayed far too long due to technological advancements. The Librephone project marks a significant step towards achieving this goal.
