# Free software scares normal people—Daniel De Laney

# Free software can be frustrating for normal people

As someone who helps friends and family with computer-related tasks, I've learned which computing tasks are particularly challenging for normal people.

Normal people often struggle with converting videos, requiring them to watch, upload, or otherwise interact with the file in various ways. This format issue is common and affects many users, making it hard for free software developers to create tools that cater to average computer users.

# Why does this happen?

The Free Software Foundation (FOSS) development team often creates powerful software solutions that have a "power user" interface, alienating those who require easier-to-use options. As a result, some people give up on using free software altogether or ask others like myself to do their work for them.

## The Magicbrake solution

To address this issue, I've created a simple front-end called Magicbrake that hides the power and flexibility of popular tool Handbrake. In just one button, you can take a video file in an unconvertible format (MP4) and transform it into a standard file.

## The appeal of streamlined tools

Magicbrake is designed to answer common questions like:

* Am I still able to use Handbrake? If not, this magic solution makes it work.
* Can someone change the default settings without modifying Handbrake?
* What edge case might exist in the Free software world?

## The benefits of streamlined tools

By removing cumbersome options and focusing on what matters most – converting videos to a common format – we can:

Help people get more done with less friction
Reduce wasted time spent learning new software or troubleshooting issues
Enhance user experience, making computers faster, easier to use for everyone.
