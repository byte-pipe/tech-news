# Removing obfuscation in Java Edition | Minecraft

**Removing Obfuscation in Minecraft: Java Edition**

**What is Obfuscation?**
Minecraft: Java Edition has been obfuscated since its release, meaning that some parts of the code are hidden or scrambled to prevent others from being modified.

**A Brief History**
The obfuscation technique was used as a common practice in the gaming industry, making it difficult for modders and developers to easily understand and modify the game. In 2019, Minecraft: Java Edition released "obfuscation mappings" which helped make the code more understandable, but this still left room for improvement.

**Removing Obfuscation**
Starting with the first snapshot following the Mounts of Mayhem launch (March 1st, 2025), Minecraft: Java Edition will no longer obfuscate its code. This means that all original names in the code will now be included by default to facilitate modding.

**Why are we doing this?**
Modding is an essential part of the Minecraft community. By removing obfuscation, modders can:

* Create and improve mods faster
* Debug crashes and crashes logs more easily
* Take advantage of de-bugging features
* Work together in a collaborative community

We hope that this change will pave the way for a brighter, more developer-friendly future of Minecraft: Java Edition.
