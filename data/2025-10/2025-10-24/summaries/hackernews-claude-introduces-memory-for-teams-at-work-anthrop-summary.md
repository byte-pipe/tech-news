# Claude introduces memory for teams at work \ Anthropic

# Bringing Memory to Claude

Claude is introducing a new feature called memory in its latest update. Here are the key points and essential details about this change:

## What does memory do?

Memory allows Claude to store and recall context from your previous conversations, projects, and team settings in one place.

## New Features

*   **Project-Specific Memories**: Project memories for each project allow you to keep track of specific work-related items, such as client needs, project details, and priorities.
*   **Team Context**: Memory remembers your team's processes, products, and initiatives, enabling Claude to understand the context behind conversations with clients and other teams.
*   **Confidential Discussions**: Project boundaries ensure that confidential discussions remain separate from general operations.

## How does memory work?

1.  To enable memory in settings:
    *   Follow these steps: In Settings, you can choose not to use memory and select whether to start using it immediately.
2.  Using memory:
    *   Create a new project memory by selecting "Memory from [project name]" in the chat menu.
    *   Claude will create a separate storage for this information.

## Additional Information Regarding Memory

*   In settings, you can see what memories you currently have from Claude (Section 1)
*   You can update any item that was previously created with your memory by chatting with Claude.
