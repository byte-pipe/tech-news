# GitHub backs down, kills Copilot pull‑request ads after backlash

## Background
- GitHub added a feature that lets Copilot insert “tips” (promotional messages) into any pull request where the model is mentioned.  
- An example surfaced when Australian developer Zach Manson asked Copilot to fix a typo; the PR then displayed a Raycast promotion with a lightning‑bolt emoji.  
- Over 11,400 pull requests were found containing similar tips, many inserted by Copilot itself.

## Developer reaction
- Manson described the inserted tip as offensive and unexpected, noting he was unaware Copilot could edit other users’ PR descriptions and comments.  
- The broader developer community viewed the tips as unwanted advertisements and a misuse of Copilot’s editing capabilities.

## GitHub’s response
- After the issue was reported on Monday, GitHub VP of Developer Relations Martin Woodward acknowledged the problem.  
- Tim Rogers, principal product manager for Copilot, explained the original intent was to help developers discover new workflow features, but community feedback showed the decision was a “wrong judgement call.”  
- GitHub disabled the tip‑insertion feature for pull requests created or touched by Copilot, stating the behavior will no longer occur.

## Official statements
- Woodward: “GitHub does not and does not plan to include advertisements in GitHub. We identified a programming logic issue with a GitHub Copilot coding agent tip that surfaced in the wrong context within a pull request comment. We have removed agent tips from pull request comments moving forward.”  
- Rogers confirmed the feature is disabled and will not be re‑enabled.

## Timeline
- **Monday morning:** Manson’s report spreads via tech news outlets.  
- **Monday afternoon:** GitHub announces the rollback of the tip feature and disables it for all pull requests.

## Implications
- The incident underscores concerns about AI models modifying user‑generated content without explicit consent.  
- It highlights the need for clear boundaries and transparent controls when integrating AI assistance into developer workflows.
