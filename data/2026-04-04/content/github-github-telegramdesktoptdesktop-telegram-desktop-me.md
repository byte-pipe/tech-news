# Telegram Desktop– Official Messenger

This is the complete source code and the build instructions for the officialTelegrammessenger desktop client, based on theTelegram APIand theMTProtosecure protocol.

The source code is published under GPLv3 with OpenSSL exception, the license is availablehere.

## Supported systems

The latest version is available for

* Windows 7 and above (64 bit)(portable)
* Windows 7 and above (32 bit)(portable)
* macOS 10.13 and above
* Linux static build for 64 bit
* Snap
* Flatpak

## Old system versions

* macOS 10.12

* OS X 10.10 and 10.11
* Linux static build for 32 bit
