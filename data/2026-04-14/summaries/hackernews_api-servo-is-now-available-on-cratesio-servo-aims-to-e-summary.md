# Servo is now available on crates.io

## Key announcements
- v0.1.0 of the `servo` crate has been released, allowing Servo to be used as a library.
- The demo browser `servoshell` will not be published on crates.io.
- Release process has matured over five releases since the initial GitHub launch in October 2025; the main bottleneck is the monthly blog post, which will be delivered in the coming weeks.

## Versioning and roadmap
- This is not a 1.0 release; the definition of a 1.0 version for Servo is still being discussed.
- The increased version number reflects growing confidence in Servo’s embedding API and its ability to meet user needs.

## Long‑term support (LTS)
- An LTS version of Servo is offered to accommodate embedders who prefer scheduled half‑yearly major upgrades.
- LTS provides security updates and, when possible, migration guides, even though regular monthly releases may contain breaking changes.
- More information on the LTS release can be found in the corresponding section of the Servo book.
