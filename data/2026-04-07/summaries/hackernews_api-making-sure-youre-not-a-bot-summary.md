# Making sure you're not a bot!

## What is Anubis?
Anubis is a software protection tool set up by websites to prevent scraping efforts from aggressive AI companies, which can cause downtime and resource unavailability.

### How does it work?
Anubis uses the Proof-of-Work (PoW) scheme similar to Hashcash, designed for reducing email spam. However, its effectiveness depends on individual scales vs. mass levels of scraper activity. In this case, its real purpose is to create a "good enough" solution where more time can be spent on fingerprinting and identifying headless browsers.

## Key points:

* Anubis prevents AI companies from scraping websites.
* It uses JavaScript features that may not work with plugins like JShelter.
* Anubis version 1.21.3 is currently available.
* For most users, JavaScript needs to be enabled for safe access.

Note: Anubis cannot completely prevent scraping efforts and does not offer a genuine proof-of-work solution. Its existence is an attempt to mitigate the harm caused by aggressive AI companies.
