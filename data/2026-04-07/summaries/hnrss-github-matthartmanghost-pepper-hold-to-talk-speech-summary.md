**Summary**

This article is about a speech-to-text app called Ghost Pepper that allows users to record their voice and then transcribe it into any text field using hold control, release and paste features. The app uses open-source models powered by WhisperKit and LLM (Large Language Model) from Hugging Face. It's designed for macOS 14.0+, Apple Silicon (M1+), and is 100% local, meaning no cloud APIs or data are sent to the user's machine.

## Key Points

* Ghost Pepper allows users to record their voice using hold control and release and paste features
* Runs entirely on the Mac, making it 100% local
* Uses open-source models powered by WhisperKit and LLM from Hugging Face
* Supports multi-language typing with models in languages such as English, French, Spanish, Chinese, Japanese, Korean, Arabic, Russian, Hindi, and many more
* Features include menu bar app, customizability for cleanup prompt, self-correction, and more

## How it Works

Ghost Pepper uses its built-in audio capabilities to record the user's voice. When the user speaks, their voice is recorded in a format that can be transcribed into text by the models. The models are then used to clean up the transcription by removing filler words and self-corrections.

**History and Releases**

Ghost Pepper has been released for macOS 14.0+. The app allows users to build it from source and get started quickly, with grant of Microphone and Accessibility permissions built-in.
