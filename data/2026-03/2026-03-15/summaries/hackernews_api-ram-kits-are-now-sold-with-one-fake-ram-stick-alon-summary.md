# RAM kits are now sold with one fake RAM stick alongside a real one to create a performance illusion — 1+1 “value pack” offers desperate psychological relief as the memory shortage worsens

## Main announcement
- V‑Color introduced “1+1” DDR5 value packs that pair one genuine memory module with a matching dummy (filler) module.
- Packs are aimed at AMD platforms and are sold as a cost‑effective way to obtain the visual look of a dual‑module setup when budget only allows a single real stick.

## Product details
- Two series are launched: **Manta Sky** (16 GB) and **Manta XFinity** (24 GB).
- Both start at DDR5‑6400 speed; timing and XMP 3.0 support were not disclosed, but AMD EXPO support is expected.
- Future “2+2” configurations are planned, offering two real sticks plus two filler sticks for enhanced aesthetics.

## Context and motivation
- Global DRAM shortage has driven RAM prices to historically high levels, putting high‑performance memory out of reach for many builders.
- Dummy modules, first popularized by Corsair’s Light Enhancement Kits, serve only cosmetic purposes: they light up and sync with RGB but add no capacity or performance.

## Performance considerations
- Single‑channel operation with a real + dummy pair can work, but dual‑channel remains the performance optimum.
- AMD Ryzen CPUs with 3D V‑Cache (e.g., Ryzen 7 9850X 3D, Ryzen 7 9800X 3D) can mitigate some single‑channel penalties, though a measurable loss remains versus true dual‑channel.
- The impact varies by workload; memory‑intensive games and applications will feel the regression more strongly.

## Availability and pricing
- V‑Color has not released pricing or exact launch dates.
- Distribution will be through selected global partners such as Newegg.
