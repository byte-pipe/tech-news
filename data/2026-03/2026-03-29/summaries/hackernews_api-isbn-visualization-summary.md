# How to Enable JavaScript in Your Browser

## Common Browsers
- **Google Chrome**
  - Open Settings → Privacy and security → Site Settings → JavaScript.
  - Toggle the switch to “Allowed (recommended)”.
- **Mozilla Firefox**
  - Type `about:config` in the address bar and press Enter.
  - Search for `javascript.enabled` and set it to `true` by double‑clicking.
- **Microsoft Edge**
  - Go to Settings → Cookies and site permissions → JavaScript.
  - Turn on the “Allowed” option.
- **Safari (macOS)**
  - Open Safari → Preferences → Security.
  - Check the box “Enable JavaScript”.

## Mobile Browsers
- **Chrome for Android**
  - Tap the three‑dot menu → Settings → Site settings → JavaScript.
  - Enable the toggle.
- **Safari on iOS**
  - Open Settings app → Safari → Advanced.
  - Switch on “JavaScript”.

## Verifying JavaScript Is Active
- Visit a site like `https://www.enable-javascript.com/` to confirm that JavaScript is running correctly.

## Troubleshooting
- Clear browser cache and reload the page if the message persists.
- Ensure no extensions or security software are blocking JavaScript execution.
