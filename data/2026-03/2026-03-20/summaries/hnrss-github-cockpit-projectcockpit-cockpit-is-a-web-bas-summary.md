**Cockpit Project Overview**

* **Overview**: The Cockpit project provides a web-based graphical interface for servers, offering a lightweight and interactive server admin console.
* **Features**: Easy to use; interacts directly with Linux sessions in browsers; supports various Linux distributions (Debian, Fedora, RHEL); and enables real-time logging and system administration tasks.
* **How it works**:
	+ Users install Cockpit on their Linux systems via package manager.
	+ A web-based interface is launched using Cockpit, with jump buttons to access the terminal or SSH connections.
