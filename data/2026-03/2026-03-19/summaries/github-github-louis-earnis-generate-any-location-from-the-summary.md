**Arnis Minecraft World Generation Tool**

### Overview
Arnis creates complex and accurate Minecraft Java Edition (1.17+) and Bedrock Edition worlds from real-world geography, topography, and architecture.

### Key Features

* Processes geospatial data from OpenStreetMap and elevation data to create detailed Minecraft representations of terrain and architecture
* Handles large-scale geographic data and generates maps with ease
* Supports map generation on mobile or larger scales
* Compatible with both Java Edition (1.17+) and Bedrock Edition worlds

### Usage

* Download the latest release or compile the project locally using the **Start Generation** feature
* Use the rectangle tool to select a Minecraft world area, then click start to generate the corresponding world
* Customize generation settings, such as scale, spawn point, or building interior generation
