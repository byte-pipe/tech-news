# GitHub - andrewyng/context-hub · GitHub

**GitHub - Context-Hub (GitHub Repository)**

* **Purpose**: Provides context for coding agents by offering curated documentation, API keys, and interactive tools.
* **Features**:
 + Curation of Markdown documents on various topics, including AI-related information
 + Interactive search functionality via CLI command `chub search`
 + API endpoint for retrieving Python documentation using the `chub get` command
 + Support for JavaScript documentation with the `chub get` command in that language
* **Commands**:
 + `chub search <query>`: Search through available content and return relevant results
 + `chub get <id>`: Fetch a specific document or skill by ID, where `<id>` can be used to reference previously opened IDs
+ ` annotat...` (multiple): Use the CLI command to annotate documents or skills with notes; also offers an option to clear annotations: `annotat--clear`
* **Notes**:
 + The context hub uses coding agents and is mainly intended for self-improving agents, not direct user interaction. It provides resources and tools that can help agents learn and improve.
 + New content types, including agent skills and community feedback, are being developed as part of the roadmap.
