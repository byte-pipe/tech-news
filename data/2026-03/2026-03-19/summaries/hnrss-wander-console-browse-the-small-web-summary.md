# Wander Console - Browse the Small Web

**Wander Console: A Web-Based Browsing Experience**
=====================================================

*   To use this application, simply **leave your current console** and access a new console from a different website.
*   The web-based console allows you to **browse random websites** and navigate the Wander network recursively.

### Benefits of Using the Wander Console

*   **Flexibility**: You can switch consoles if preferred or explore other websites' directories within the Wander community.
*   **Personalization**: As a member of the Wander community, you have access to recommendations from other consoles linking back to your own directory.

### Setting Up Your Own Wander Console
----------------------------------------

1.  Download the required ZIP file and extract `index.html` and `wander.js`.
2.  Place the extracted files in the `/wander/` directory of your website.
3.  Edit `wander.js` according to [the codeberg.org repository](https://codeberg.org/susam/wander) directions.
