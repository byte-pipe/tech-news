# ArrowJS — The first UI framework for the agentic era  

## Overview  
- ArrowJS is a tiny, zero‑dependency, type‑safe UI framework that requires no build step.  
- It can isolate component logic in WebAssembly sandboxes and render inline DOM directly (no iframes or pre‑defined components).  
- Designed for fast, maintainable applications and on‑demand UIs generated by chat agents.  

## Core Concepts  
- **Reactive primitives**: only three functions are needed – `reactive`, `html`, and `component`.  
- Built on plain JavaScript/TypeScript primitives (modules, functions, template literals).  
- Documentation fits in less than 5 % of a 200 k token context window.  

## Key Features  
- No JSX, no React/Vue compilers, no Vite plugins (optional SSR plugin available).  
- Runtime size under 5 KB over the wire.  
- Works with the Arrow sandbox for secure, sandboxed UI generation.  
- Full TypeScript support and editor integration (VSCode syntax extension).  

## Quickstart & Installation  
- Scaffold a Vite 8 Arrow app with SSR, hydration, and routing in one command:  

  ```bash
  pnpm create arrow-js@latest arrow-app
  ```  

- Optional coding‑agent skill wrapper:  

  ```bash
  npx @arrow-js/skill@latest
  ```  

- Core runtime can be added without a build tool:  

  - From npm: `npm install @arrow-js/core`  
  - From a CDN:  

    ```html
    <script type="module">
      import { reactive, html } from 'https://esm.sh/@arrow-js/core';
    </script>
    ```  

## Reactive Data  
- `reactive(value)` wraps objects/arrays to create observable state.  
- `reactive(() => expr)` creates a computed value that re‑evaluates when its tracked reads change.  
- State can be used for local component data, shared stores, or mutable props.  
- Manual subscriptions are available via `$on` and `$off`.  

## Templates  
- Use the tagged template literal `html\`...\`` to define mountable templates.  
- Expressions are static by default; passing a function (`${() => data.foo}`) makes the slot reactive.  
- Supports rendering of text, attributes, properties, lists, nested templates, and events.  

### Reactive Attributes Example  
```ts
import { html, reactive } from '@arrow-js/core';

const data = reactive({ disabled: false });

html`
  <button disabled="${() => data.disabled}">Save</button>
`;
```

ArrowJS offers a minimalist, type‑safe, and ultra‑fast approach to building reactive UIs, leveraging only native TypeScript functions and template literals.
