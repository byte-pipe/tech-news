# Claude Code isn’t going to replace data engineers (yet)

## What's the Difference Between Using dbt and Claude Code?

**dbt Summary**
### Introduction to DBT

dbt stands for Data Builders Tool, a data integration platform that automates data refresh processes. It is commonly used by data engineers to manage their workflows.

### Using dbt

DBT provides various pre-built connectors, models, and APIs to connect databases from different vendors. This allows users to easily import, transform, and load complex datasets into various storage systems.

**Claude Code Summary**

Claude code is a Python script based on dbt that automates the process of building data pipelines for specific industries or domains. It uses natural language processing (NLP) techniques to discover patterns in data and create optimized workflows.

**Key Points:**

* Claude code can build credibile DBT projects from real data
* Claude needs API source, not just analysis work done beforehand
* Skill sets provided by dbt labs affect the quality of the database
