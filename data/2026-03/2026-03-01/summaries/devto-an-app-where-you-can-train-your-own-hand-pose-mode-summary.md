# An App where you can Train your Own Hand Pose Model for your Project! 🤌

## The Community
- Aimed at the AI/ML community, especially beginners who want to create their own models.
- Highlights the scarcity of free tools that let users train, download, and integrate models without signing up or paying.
- Points out limitations of pre‑trained models and the difficulty of hiding API keys in projects like Chrome extensions.

## What I Built
- Developed a website called **HandTracker** that uses ml5.js Hand Pose to let users train custom hand‑gesture models.
- Users can define any gesture (e.g., “Hello”, Gojo’s signature) and assign it to a class label.
- The tool is free, allows model download, and requires no external accounts.

## Demo & Features
- Live demo: https://francistralt.github.io/HandTracker/ (camera must be enabled).
- Settings panel includes:
  1. **Define Classes** – map gestures to class names.
  2. **Collection** – set the number of samples per class (recommended equal across classes).
  3. **Training Parameters** – adjust based on sample size.
  4. **Begin Training** – starts training; pauses automatically when the hand is not visible.
- After training, users can test the model in‑browser.
- Model can be downloaded as a zip containing `model.json`, `model.weights.bin`, and `model_meta.json`.

## How I Built It
- Implemented with vanilla HTML, CSS, and JavaScript.
- Utilized ml5.js for machine‑learning operations and p5.js for supporting graphics.
- All code was generated by Google Gemini, with Gemini handling 100 % of the implementation.
