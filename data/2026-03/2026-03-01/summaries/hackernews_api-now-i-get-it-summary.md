# Now I Get It!

- The service transforms scientific PDFs into shareable, interactive web pages that explain the content in plain language.
- Users can upload a PDF (preferably under 10 MB) via drag‑and‑drop or file browser.
- The system processes the file through several steps: security check, classification, reading the paper, generating the interactive page, and publishing it online.
- After generation, a link to the interactive page can be copied for easy sharing.
- The interface displays progress indicators such as “Uploading PDF,” “Security check & classification,” “Reading paper,” “Generating interactive page,” and “Publishing to web.”
