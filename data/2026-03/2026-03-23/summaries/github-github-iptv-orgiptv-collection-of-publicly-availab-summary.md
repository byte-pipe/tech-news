# GitHub - iptv-org/iptv: Collection of publicly available IPTV channels from all over the world · GitHub

### IPTV Collection
=====================

The **iptv-org** GitHub repository provides a collection of publicly available IPTV (Internet Protocol television) channels from all over the world.

### Key Points
----------------

*   The repository contains links to various IPTV playlists, including main playlists and individual channel streams.
*   For easy access, these playlists can be found at `https://iptv-org.github.io/iptv/index.m3u`.
*   Additional resources, such as EPG (Electronic Program Guide) files for most of the channels and API documentation, are also available.

### Use Instructions
--------------------

1.  **Paste Playlist Link**: Simply paste a link to one of the playlists into any video player that supports live streaming and press Open.
2.  **Get EPG Data**: Links to other playlists can be found in the `PLAYLISTS.md` file.
3.  **Explore Database**: All channel data is taken from the `iptv-org/databaserepository`.
4.  **Check API Documentation**: The API documentation can be found in the `iptv-org/apirepository`.

### What Is Included
----------------------

*   Public IPTV playlists with links to individual channels and programs.
*   Electronic Program Guide (EPG) files for most of the channels.
*   Database containing channel data from the website.

### How It Works
--------------

When you paste a playlist link, a new episode is generated based on that link. This allows users to access IPTV content without having to manually search for programs or subscriptions.

By making public links available, **iptv-org** provides viewers with easy access to various international channels and events, while ensuring compliance with copyright laws and respecting the intellectual property of content creators.
