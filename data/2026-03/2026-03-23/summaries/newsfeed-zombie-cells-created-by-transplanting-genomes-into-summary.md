# 'Zombie' cells created by transplanting genomes into dead bacteria | New Scientist

**Creating Living, Synthetic Bacterial Cells**

* A living, synthetic bacterial cell has been made by transplanting a complete genome into a dead bacterium.
* The breakthrough came after researchers successfully killed the host cells first and then added a synthetic version of another bacterium's genome to revive them.
* The new cells, called "zombie cells," were able to grow and divide normally with genetic tests showing they carried the synthetic genome.

**Key Points:**

* Researchers successfully transplanted genomes into dead bacteria for the first time.
* The process involved killing host cells first to ensure the cell was not relying on its original genome.
* The new cells, or "zombie cells," can now be used to create genetically modified organisms with desirable traits.
* This breakthrough marks a significant step forward in synthetic biology.
