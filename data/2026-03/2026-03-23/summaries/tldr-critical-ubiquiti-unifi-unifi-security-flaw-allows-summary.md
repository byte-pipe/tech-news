## Ubiquiti Fixes Two Vulnerabilities in UniFi App, Including a Critical Flaw That Could Enable Account Takeover

* ** vulnerably-exposed software used by thousands*: *
The Ubiquiti UniFi Network app has been found to be vulnerable to exploitation, allowing potential account hijacking.

### Key Details:

*   The vulnerabilities can be exploited using social engineering techniques and have the potential to allow an attacker to take over a user's account.
*   A maximum severity issue (CVE-2026-22557) affects UniFi Network application version 10.1.85 or earlier, while another vulnerability (CVE-2026-22558) has been addressed in versions 10.1.89 and later.

### Possible Impact:

*   An attacker with access to the network can exploit these vulnerabilities to gain control over sensitive data.
*   This could include unauthorized access to system files, manipulation of underlying account settings, or even the ability to elevate privileges.

### Remediation:

*   The second vulnerability found in UniFi Network app 10.1.89 was addressed by Ubiquiti.

### Conclusion:

The Ubiquiti UniFi Network app has been compromised due to two newly identified vulnerabilities that could be exploited for potential account hijacking. IT administrators should check and address these issues immediately, especially if they have users relying on this software.
