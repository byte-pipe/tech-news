## Installation guides

### Windows
- **Zip**: download `nvim-win64.zip` (or `nvim-win-arm64.zip`), extract, run `nvim.exe`.  
- **MSI**: download `nvim-win64.msi` (or `nvim-win-arm64.msi`), install, run `nvim.exe`.  
- Note: Windows Server may require `vcruntime140.dll`.  

### macOS
- **x86_64**: download `nvim-macos-x86_64.tar.gz`, clear quarantine with `xattr -c`, extract, run `./nvim-macos-x86_64/bin/nvim`.  
- **arm64**: same steps using `nvim-macos-arm64.tar.gz`.  

### Linux (x86_64)
- **AppImage**: download `nvim-linux-x86_64.appimage`, make executable (`chmod u+x`), run; if FUSE is unavailable, extract with `--appimage-extract` and run the binary inside `squashfs-root`.  
- **Tarball**: download `nvim-linux-x86_64.tar.gz`, extract, run `./nvim-linux-x86_64/bin/nvim`.  

### Linux (arm64)
- Mirrors the x86_64 instructions using `nvim-linux-arm64.appimage` and `nvim-linux-arm64.tar.gz`.  

### Other methods
- Installation via system package managers is also supported.
