* Emphasizes GPU-based computation
* Integrates two backend modules: `warp.simmodule` and `muJoCo Warp`

**System Requirements**

* Python 3.10+
* Operating System: Linux (x86-64, aarch64), Windows (x86-64), or macOS (CPU only)
* NVIDIA GPU with Maxwell or newer driver 545 or newer (CUDA 12)
