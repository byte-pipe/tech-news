# Introducing Unsloth Studio | Unsloth Documentation

**Unsloth Studio Beta Launch**

### Overview
Unsloth Studio is an open-source, no-code web UI for training, running, and exporting open models in one unified local interface. It supports a wide range of AI tasks, including text, vision, TTS audio, embedding models.

### Key Features

* Run GGUF (Gradient Boosting via Efficient computation) and safetensor models locally on Mac, Windows, Linux.
* Train 500+ models up to 70% faster and with 70% less VRAM without accuracy loss.
* Support text, vision, TTS audio, and embedding models training.
* MacOS and CPU work for ChatGPT inference and MLX training.
* No dataset needed, auto-create datasets from PDF, CSV, JSON, DOCX, TXT files.
* Export or save models to safetensor, 16-bit safe Tensor, or GGUF.
* Self-healing tool with web search, auto-inference parameter tuning, code execution, and APIs (coming soon).
* Auto inference parameter tuning and edit chat templates.

### Features by hashtag

**#Run models locally**: Explore GGUF and safetensor models using self-healing tools calling, web searching, auto-inference parameter tuning, code execution, and APIs.
**#No-code training**: Upload PDF, CSV, JSON documents and start training instantly on NVIDIA. Optimized LoRA, FP8, FFT, PT for multi-GPU inference.
**#Data Recipes**: Transform documents into useable/synthetic datasets via graph-node workflow with NVIDIA Data Designer.

### Features by hashtag

**#Observability**: Track training loss, gradient norms, and GPU utilization in real-time and customize to your liking using unsloth's tracking logs.
**#Export models**: Export any model type to safetensor or GGUF for use with llama.cpp, vLLM, Ollama, LM Studio, and more.
**#Model Arena**: Compare two different models and see how their outputs differ. Load one first, then the second, and compare inference.

### Features by hashtag

**#Privacy first + Secure**: Unsloth Studio is 100% offline and locally on your computer, with token-based authentication for seamless data security.
