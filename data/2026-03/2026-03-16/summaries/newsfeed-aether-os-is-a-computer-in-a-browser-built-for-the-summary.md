# Aether OS: A Computer in a Browser Built for the AT Protocol

Aether OS is a computer in a browser that ties directly into the Bluesky at-scale protocol. It offers a suite of apps, including text editing and task management, with 42 extensions covering social media and other functions. The app is built on top of AetherKit and utilizes the AT Protocol to encrypt data offline.

**Key Features:**

*   A full-fledged desktop in a browser
*   Integrated suite of apps for various tasks
*   Integration with public records through Bluesky account
*   Cyberpunk-inspired design

**Rough Edges and Challenges:**

*   The project is still in alpha, with documentation incomplete
*   Users may experience issues with app functionality due to lack of documentation or encryption concerns
