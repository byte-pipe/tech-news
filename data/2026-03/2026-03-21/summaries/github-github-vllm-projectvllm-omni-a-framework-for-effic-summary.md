# GitHub - vllm-project/vllm-omni: A framework for efficient model inference with omni-modality models · GitHub

Here is a concise and informative summary of the article:

**vllm-project/vllm-omni**

This is a framework for efficient model inference with omni-modality models, specifically designed to serve large language models for text-based tasks. **Omni-Modality Models**: The project focuses on supporting a wide range of data types (text, image, video, audio) and architectures (non-autoregressive).

**Key Features:**

* Omni-modality model inference and serving
* Text, image, video, and audio processing capabilities
* Non-autoregressive architecture support

**Contributions:**

* Omni-modality model skills are available for developers to use in their coding assistants
* A wide range of omni-models can be used with popular AI coding tools (Cursor IDE, Claude, Codex)
