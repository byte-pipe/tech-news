# GitHub - paniclock/paniclock: Instantly disable Touch ID and lock your Mac with one click or keyboard shortcut. · GitHub

Here is a concise and informative summary of the article "paniclock" by preserving its grammatical person, narrative style, and point of view:

**Overview**

PanicLock is a macOS utility that instantly disables Touch ID and locks the screen with one click or closing your laptop lid.

**Key Features**

* One-click panic lock: Disables Touch ID and locks the screen
* Lock on Close: Optionally locks and disables Touch ID when you close the lid
* Temporarily disables Touch ID: Forces password-only unlock
* Auto-restore: Restores original Touch ID settings after unlock
* Keyboard shortcut: Configure a global hotkey (e.g., ⌃⌥⌘L)
* Launch at login: Starts automatically when you log in

**Installation and Requirements**

* macOS 14.0 (Sonoma) or later with Touch ID
* Homebrew installation: `brew install paniclock/tap/paniclock` or manual download from the DMG file

**Usage Guide**

1. Trigger panic lock immediately by clicking the "icon" button.
2. Configure lockdown options in Preferences, including Lock on Close and Auto-restore.

**Benefits and Limitations**

PanicLock provides a convenient solution for sensitive situations where Touch ID is required, but biometrics may not be suitable. However, its installation requires administrative privileges, which may cause issues when logging out or shutting down.
