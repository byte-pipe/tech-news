# Congrats to the Notion MCP Challenge Winners! - DEV Community

**Notion MCP Challenge Winners: NoteRunway, DevNotion & Relay**

The winners of the NoteOnion MCP (Management Collaboration Platform) Challenge have been announced, and they are:

* **NoteRunway**: Giorgi Kobaidze
* **DevNotion & Relay**: Yash Kumar Saini and Christian Apostol

**Summary**

The Notion MCP Challenge was an AI-powered workspace and GitHub automation contest. The contest attracted entries from developers who showed creativity and technical depth in how to leverage NoteOnion MCP for meaningful solutions.

**Key Points:**

* Three winners were selected for the challenge: NoteRunway, DevNotion & Relay
* These winners created various Notion-based tools and automations that leveraged NoteOnion MCP's capabilities

**The Winners' Tools and Solutions:**

1. **NoteRunway**: An AI-powered workspace management tool with a hybrid architecture using direct API access for bulk reads and not-as-a-safety-layer.
2. **DevNotion & Relay**: Automated GitHub activity harvesting, blogging, and incident management assistant solutions that leveraged the capabilities of NoteOnion MCP.

**What the Winners Did:**

* Giorgi Kobaidze built an AI-powered workspace manager with an API-driven approach.
* Yash Kumar Saini and Christian Apostol created an automated GitHub activity pipeline and blog post generator for DEV.to.
