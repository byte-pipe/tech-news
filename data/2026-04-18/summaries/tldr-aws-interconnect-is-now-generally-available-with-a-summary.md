# AWS Interconnect is now generally available, with a new option to simplify last-mile connectivity | AWS News Blog

## AWS Interconnect is Now Generally Available

AWS has announced the general availability of **AWS Interconnect**, a managed private connectivity service that simplifies last-mile connectivity to AWS from various cloud providers.

### Key Highlights

- **Multicloud Connectivity**: Connects your AWS environment with other cloud providers through Google Cloud, Microsoft Azure, and Oracle Cloud Infrastructure (OCI) starting in 2026.
- **Last-Mile Connectivity**: Establishes high-speed private connections between AWS and on-premises networks through existing network providers without managing infrastructure complexity.

### Benefits

- Simplification through a managed services experience
- Predictable latency and consistent throughput
- Segregated security management by default, with each cloud provider managing encryption independently

### Configuration

AWS Interconnect can be configured in the **AWS Console** for ease of use, selecting locations, partners, regions, and bandwidth requirements.
