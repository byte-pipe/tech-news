# CadQuery | Create parametric CAD models with Python

## CadQuery Overview

CadQuery is a powerful open-source Python library that enables building 3D CAD models by describing them as code. This allows for easier model management, sharing, and parameterization without the need for a graphical user interface (GUI).

### Main Features

*   Describes parts as code for model creation
*   Easy to version, share, and parameterize models
*   No GUI required, making it suitable for collaborative projects

## CadQuery Documentation

The CadQuery documentation provides detailed information on the library's capabilities, installation instructions, and user guidance.

### Downloads

The downloadable libraries can be accessed through pip:

```bash
pip install cascadquery
```

This command will provide a set of Python modules that you can use to interact with the CadQuery API.
