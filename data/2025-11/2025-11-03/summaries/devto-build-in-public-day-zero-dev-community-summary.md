# Build in Public: Day Zero - DEV Community

**Building in Public for Real-World Impact**

### The Problem

*   Outdated influencer spreadsheets and questionable databases
*   Priced agencies with startup-friendly fees
*   DIY approaches resulting in overpaid professional services or extensive manual effort
*   Inefficient process relying on manual data collection from various platforms

**The Challenge**

Creating a web-based platform to help users identify relevant voices and communities online, especially influencers, in a specific niche (e.g., small businesses in Barcelona).

### The Solution

We aimed to create an open-source tool that utilizes Bright Data's public profile data feed and artificial intelligence models to produce meaningful summaries. To address the manual input challenge, our platform features:

-   **Automated Public Profile Data Collection** through interactive APIs.
-   **AI-powered Analysis**, generating relevant insights based on the user-inputted query.

### Early Lessons

*   The AI model's accuracy relies heavily on high-quality public profile data ( Bright Data's feed).
*   Manual data collection is unnecessary, as our platform can automate the process with significant efficiency gains.
