# GitHub - traefik/traefik: The Cloud Native Application Proxy

**traefik Cloud Native Application Proxy Summary**
=====================================================

* The **traefik** application is a cloud-native web proxy, monitoring, and security platform.
* It provides automated service discovery, load balancing, and SSL termination.
* Available as open-source software under the MIT license.

### Features

* Monitors applications for errors or anomalies
* Allows notifications to be set up through the **traefik.io** portal
* Integrates with different versions of Docker, Kubernetes, and other containerized systems.
