## PaddlePaddle/PaddleOCR Documentation

### Overview
PaddlePaddle/PaddleOCR is a powerful, lightweight OCR (Optical Character Recognition) toolkit that enables users to convert PDF or image documents into structured data for their AI models.

### Key Features

* Bridges the gap between images/PDFs and LLMs (Large Language Models)
* Supports 100+ languages
* Publicly available at [www.paddleocr.ai](http://www.paddleocr.ai)
