# GitHub - traefik/traefik: The Cloud Native Application Proxy

**Summary**

The GitHub repository `traefik/traefik` provides a Cloud Native Application Proxy (CNAP) for Kubernetes, a popular container orchestration tool.

**Key Points:**

* The CNAP is intended for deployment public-facing applications.
* To access the application, navigate to [Traefik](https://docs.traefik.io/en/v1.9/home/).
* You must be logged in as an administrator to change notification settings.
* The latest version of Traefik is 0.11 (5.6k stars).
