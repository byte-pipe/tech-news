# PaddlePaddle/PaddleOCR Overview
=====================================

## Introduction

PaddlePaddle is a powerful, lightweight OCR (Optical Character Recognition) toolkit that connects images/output PDF documents with Large Language Models (LLMs). The tool supports 100+ languages and provides structured data for AI applications.

## Key Features

### Key Benefits:

*   Bridges the gap between images/PDFs and LLMs
*   Supports 100+ languages
*   Designed to be lightweight and efficient

### Code Overview

PaddlePaddle is a Python library that can be forked or starred. The repository has around 63.8k stars on GitHub.
