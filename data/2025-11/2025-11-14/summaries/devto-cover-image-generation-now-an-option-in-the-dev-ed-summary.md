# Cover Image Generation Now an Option in the DEV Editor - DEV Community

** Prompt-Driven Image Generation Introduced in DEV Editor**

The community is excited to announce the introduction of a new feature in the DEV editor: prompt-driven image generation. This innovative tool allows users to generate images directly within the post editor with ease. Here's an overview of how it works and its benefits.

### Key Features & Details

* **Custom Aesthetic**: The process begins with a custom "retro" system prompt that generates images with a shared aesthetic and design language. This feature is designed to align with the community's unique style.
* **Open-Source Origin**: As an open-source project, users can access the source code directly through the generation modal. This ensures transparency and encourages collaboration across the site.
* **Performance**: The image generation process is fast enough for most use cases, but rate limiting will be implemented to manage load. Real-world testing has been thorough, and we're already monitoring its performance.

### Performance and What's Next

The technology is thoroughly tested, and it loads quickly on most devices. However, we'll continue to monitor real-world usage and may adjust the limits as needed to ensure a smooth experience for all users.
