## nvm-sh Node Version Manager - POSIX-compliant bash script to manage multiple active node.js versions

### Overview
The `nvm-sh` package is a POSIX-compliant bash script that enables the management of multiple active versions of Node.js. As of my knowledge cutoff in [2023], this script provides users with the ability to easily switch between different Node.js installations and their respective environments.

### Key Features and Functionality

* Enables loading of different Node.js versions using the `--current` option
* Customizable permissions, similar to those found on Linux systems
* Provides a simple way to track multiple active installions at once

### Summary

The main features of `nvm-sh` can be summarized as follows:

- **Multiple nodejs Version Support**: Manages multiple versions of Node.js easily.

- **POSIX Compliance**: This package adheres closely to PostgreSQL compliant Linux system bash commands.

- **User Customization**: Enables users to track and switch between different versions.
