# - YouTube

# Overview of YouTube

*YouTube is a video-sharing platform that allows users to upload, share, and view videos across various devices.* It is owned by Google LLC.

## Supported Platforms

### Content Upload

Users can upload content from their personal device or the cloud.
Supported platforms:
- Personal computers
- Mobile devices
- Cloud storage services (e.g., Google Drive)

### User Interface

### Navigation

* Users can access different sections, such as:
  - Home (displaying the most popular videos)
  - Search Bar (find content related to a specific topic or keyword)
  - Menu (quick access to features and settings)
### Video Players

**Video playback occurs directly on your device using the **YouTube app.**

## Navigation

* Users can also navigate between different areas within YouTube through the following options:
 * Account page: control account settings and upload history
 * My videos: content uploaded by creators for personal viewing
   - Playlist: a collection of related videos
   - Library: stored video clips and screenshots

## Interface Features

### Video Details

* Showcases detailed information about each video, such as creation date, description, and ratings.

### Search Functionality

* Allows users to search for content using keywords or specific phrases.
* Returns a list of results, sorted by popular tags

 Users can access YouTube on their mobile devices.
