# Beat Me If You Can - DEV Community

**The Liquidcode Markdown Editor Challenge**

The current state of the Liquidcode Markdown Editor is outdated and cumbersome, but a new challenge has just been added. The goal is to build an editor that handles headings, lists, bold, italics, and all formatting options in a functional and visually appealing way.

### Solutions

**Approach 1: Classic Approach**

* Clean interface
* Live preview
* Menu bar with features
* Functional and non-distracting design

**Approach 2: Medium Clone**

* Smooth, minimal interface
* No knowledge required to use markdown
* Write naturally, convert to markdown
* Distraction-free writing experience

There are two solutions to the challenge. One of them is a new React-based editor for Liquidcode that meets specific requirements on time and execution.

**Your Turn: Beating the Founder**

Liquidcode is now opening the Markdown Editor contest, but you will face directly. Compete with a developer who has already built this platform's core features. One week from joining, submit your solution in a markdown format. Community votes on who is the winner.

The benefits of participation are:

* A unique badge for winning the challenge
* Permanent early adopter status
* Free features and lifetime perks for being one of the first participants

To find out if you're good at proofing Liquidcode's actual capabilities, take this opportunity to test your limits.
