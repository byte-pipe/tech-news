# CSS Nightmares: Building a Haunted Scene That Actually Creeped Me Out - DEV Community

**Halloween CSS Art Submission**

This is a CSS art submission for the Frontend Challenge - Halloween Edition. It features a haunted scene generated using pure HTML and CSS animations.

**Inspiration**

The inspiration behind this piece came from classic horror movies and haunted houses, evoking the feeling of being right before something jumps out. This creates an eerie atmosphere that draws the viewer in.

Watch the demonstration on our website: **Live Demo Here**

**Code & Resources**

* **Sources Code**: Pure CSS Halloween Art - a haunted night scene built entirely with HTML and CSS animations.
* **Project Structure**: halloween-css-art
	+ **index.html**: the main HTML file that loads the stylesheet and displays the scene. Each `<div>` represents an object or animated element in the scene, such as the blood moon, haunted house, spider, ghost, pumpkin, and skeleton.
	+ **halloween.css**: this stylesheet defines all visual elements and animations for the scene using techniques like positioning, gradients, and keyframes.

**Key Techniques Used**

* Positioning: Absolute layout to place each object precisely
* Gradients: used to paint every shape (no images used)
* Animations: controlled via CSS @keyframes for motion

**Built with**

This project was built purely with pure HTML and CSS. No JavaScript is required.
