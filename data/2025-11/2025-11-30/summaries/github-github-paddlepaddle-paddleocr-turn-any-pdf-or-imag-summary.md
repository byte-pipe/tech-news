# GitHub - PaddlePaddle/PaddleOCR: Turn any PDF or image document into structured data for your AI. A powerful, lightweight OCR toolkit that bridges the...

PaddlePaddle is a powerful, lightweight Omnijam OCR toolkit that enables users to convert PDF or image documents into structured data for artificial intelligence (AI) applications. Key features include:

* Brings images and PDFs together with large language models (LLMs)
* Supports over 100 languages
* Open-source under the Apache-2.0 license

To use PaddlePaddle, follow these steps:
- Fork or clone the repository at `https://github.com/PaddlePaddle/PaddleOCR`
- Build the software using Docker on Linux-based operating systems
- Install the `paddle` command
- Execute the `predict` command with input data to generate structured output for AI applications
