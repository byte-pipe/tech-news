# "2026", AI Users vs The Unemployed. - DEV Community

**Code Overload: The Rise of AI-Generated Code**

By 2026, nearly 90% of all code is predicted to be AI-generated, according to industry data. This trend has significant implications for software development, job markets, and professionals.

**The Impact on Developers**

*   Most developers use existing tools like GitHub Copilot and ChatGPT to generate code.
*   The number of lines of code generated by developers in 2024 is estimated to be around 256 billion, while 600 billion are projected in 2025. This indicates a significant increase in the automation of coding tasks.

**The Shifting Model**

*   With AI-generated code replacing traditional programming skills, many developers are facing job insecurity.
*   The focus has shifted from understanding complex concepts like data structures and algorithms to learning to work with AI tools.
*   Developers who continue to rely on human expertise may struggle in this context.

**The Human Side of Automation**

*   While AI is writing code, the actual coding tasks require human judgment and review to ensure quality and accuracy.
*   The line between senior and junior developers has become increasingly blurred by AI-generated code.
*   Professionals must adapt to a new model where AI completes some tasks, but humans handle others.

**The Future of Tech Job Openings**

*   According to recent employment data, tech job openings are down significantly due to an oversupply of qualified candidates trained on existing tools.
*   The trend towards 90% AI-generated code means that fewer professionals will be needed to create software applications.
