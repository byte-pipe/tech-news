## User Setting Preferences

### Importing Book Recommendations

*   This feature allows users to import book recommendations from external platforms that meet the site's popularity threshold.
*   The recommendation engine analyzes user reading history to provide suggestions based on their preferences.
*   A popularity score is used as a threshold, with less popular books being excluded and more popular ones being more likely to be recommended.

## Shelf Organization

### Bookshelf Structure

*   Users can select a specific shelf or create a new one to organize their favorite books.
*   The most recent 64 or fewer books in each selected shelf will be displayed prominently.
*   The "Currently Reading" section allows users to monitor the status of unread books on shelves.

## Search Functionality

### Book Search

*   Users can search for recommended books using Goodreads's built-in search functionality.
*   Only popular books with three or more ratings are included in the results.
