# Announcing AnalogJS 2.0 ⚡️ - DEV Community

**AnalogJS 2.0 Release Notes**

* Introduced Angular support based on Vite, a next-generation open-source build tool, and Nitro, an open-source server engine framework.
* Provides filesystem-based routing for efficient route handling.
* Includes markdown pages and blogs, as well as API/server routes and support for hybrid Static Site Markup (SSM) with sitemap and RSS feed integration.
* Enhances Angular CLI/Nx workspaces by supporting server and deployment modes for optimal performance.

**New Features**

* **Content Resources**: Introduces resources specifically designed to display content lists and files, optimized for efficient loading.
* **Route Meta**: Allows for personalized metadata (title and meta information) to be embedded into routes, enhancing user experience.
