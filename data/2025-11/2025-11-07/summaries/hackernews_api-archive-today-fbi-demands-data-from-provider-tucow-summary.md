# Archive.today: FBI Demands Data from Provider Tucows | heise online

## Archive.today Demands Data from Provider Tucows
### FBI Requests Data for Archiving Exposing User Privacy Concerns

The Federal Bureau of Investigation (FBI) has demanded data from a digital archiving service provider, Tucows, citing concerns about user privacy. The request is related to the recent website [Archive.today](http://archive.today), which provided an alternative solution for users concerned about online storage risks.

### Background and Issues
- The FBI launched an investigation into Tucows because of concerns over potential violations with user privacy.
- Some individuals were facing security risks due to storing large volumes of information at public destinations.

### Request Detail

The FBI requested data from the provider, which included:
  * Access logs for website users who uploaded and stored content online.
  * User database records.

The request sought specific metadata about content uploads, modifications, and access. The data collection was seen as necessary to understand how such risks affected public security and digital freedoms in general.

### Response from Tucows
- Tucows acknowledged the FBI's concerns, highlighting their strict policies regarding user data protection.
- "We respect users' right to privacy," a Tucows representative stated. "Our terms of service and acceptable use rules prioritize preserving individual information confidentiality."

Ultimately, the request was denied by authorities.
