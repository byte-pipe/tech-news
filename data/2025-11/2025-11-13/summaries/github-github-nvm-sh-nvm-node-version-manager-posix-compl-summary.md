### Purpose

The `nvm` is a tool used for managing and interacting with different Node.js versions on a system.

### Key Features and Components

*   **Public**: The package provides an API for navigating the node version hierarchy.
*   **Branch**: Supports multiple branches, such as development (`dev`) and release (`v13.0.1`).
*   **Tags**: Allows for tagging nodes with specific versions or states (active or inactive).
