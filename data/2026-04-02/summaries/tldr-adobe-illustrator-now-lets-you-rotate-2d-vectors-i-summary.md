# Adobe Illustrator now lets you rotate 2D vectors in 3D space

## Overview
- Adobe Turntable, previewed at Adobe MAX in October 2024, is now generally available in Illustrator.
- The feature lets creators rotate any vector illustration in 3D while keeping the original 2D art style.

## Key Features
- Automatically generates up to **74 editable views** from a single vector design, including full rotation and vertical tilt.
- Each angle remains a fully editable vector, allowing quick turnarounds, animations, and production‑ready assets without losing detail.
- Refined during the beta phase based on user feedback.

## Practical Applications
- Animators used Turntable to speed up character turnarounds for pitch decks.
- Game designers employed it to create concept art and multi‑angle views for new projects.
- Designers can now produce instant multi‑angle vector assets in seconds instead of hours.

## Availability
- Turntable is now part of the standard Illustrator package; no separate purchase is required.
- More information and tutorials are linked from Adobe’s official page.
