# GitHub - ytdl-org/youtube-dl: Command-line program to download videos from YouTube.com and other video sites

# YouTube Downloader Program Documentation

**Overview**

The `youtube-dl` project is a command-line program designed to download videos from YouTube.com and other video sharing sites. It provides an easy-to-use interface for users to extract audio or video content from various online sources.

**Usage**

To use `youtube-dl`, first sign in to the GitHub account associated with your YouTube channel. Once signed in, you can start downloading videos using the following command:

```bash
ytdl --quiet --out-format mp3 input.mp4
```

This will generate a single.mp3 file containing the downloaded video content.

## Features

* **Support for multiple video formats**: `youtube-dl` supports various video formats, including MP3, VMA (Video Matrix Audio), and more.
* **Customization options**: Users can modify the download behavior by adding or removing flags from the command-line arguments.
* **Sign in required**: Successful sign-in is necessary to access YouTube channels.
