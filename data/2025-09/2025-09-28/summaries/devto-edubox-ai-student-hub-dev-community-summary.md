# EduBox - AI Student Hub - DEV Community

**EduBox - AI Student Hub**
=====================================

**Overview**

EduBox is a student-centric platform that helps users manage their academic life, balancing work and study. It uses AI-powered tools to streamline organization, planning, and analytics.

**Key Features**

* **AI-driven tools**: Smart file management with semantic search; intelligent planner for avoiding last-minute cram sessions; campus life hub with club and event integration
* **Integration services**: Access to external databases and APIs; support for tool creation and customization
* **Personalized assistance**: AI assistant for answering questions, creating new assignments, and studying
* **Real-time updates**: Real-time sync across devices using Convex

**Tech Stack**

*   Frontend: Next.js + TypeScript, styled with Tailwind CSS
*   Backend: Render for serverless data operations; Vercel for seamless deployment
*   Authentication: Clerk for secure user sign-in and profile management
*   Data Storage: Convex for real-time storage and minimal latency

**System Requirements**

*   Familiarity with Next.js, TypeScript, Tailwind CSS, and Convergent
*   Basic understanding of API development and configuration

**Getting Started**

Deploy the application on Vercel to access demo features. Source code is also available.
