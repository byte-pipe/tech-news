# Notizen aus der Maschinenhalle » Blog Archive » Download responsibly!

Based on the given text, this month's message calls for users of the download server to 'download responsibly!' and asks for proactive measures from clients downloading OSM data. The author would like speedy updates to be made available earlier and the downloading experience made easier by use of a fast and reliable HTTP redirect system. As part of their appeals, the author raises three specific suggestions for optimizing downloads and avoiding network congestion - downloading everything piece-meal from a planet file hosted on the OSM project's Planet map, using pyosium-up to automate updates in a more efficient manner, and monitoring what's happening on your local file in order to build catches for better updates.
