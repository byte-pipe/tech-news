**Langfuse: Open Source LLM Engineering Platform**

Langfuse is an open source LLM (Large Language Model) engineering platform that enables teams to collaboratively develop, monitor, evaluate, and debug AI applications. It provides features for logging and tracing application functionality.

**Key Features:**

* **LLM Application Observability**: Instruments app calls, Retrieves, embeddings, and agent actions to track complex logs and user sessions.
* **Prompt Management**: Manages, versions, and collaboratively iterates on prompts using strong caching on both server and client sides.
* **Evaluations**: Adapts to the application development workflow, enabling key metrics like evaluation results.
