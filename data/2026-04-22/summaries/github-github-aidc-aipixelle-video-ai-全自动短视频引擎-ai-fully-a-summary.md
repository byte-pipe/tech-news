# GitHub - AIDC-AI/Pixelle-Video: 🚀 AI 全自动短视频引擎 | AI Fully Automated Short Video Engine · GitHub

**GitHub AIDC-AI/Pixelle-Video**

* **Overview**
 Pixelle-Vide是一个全自动短视频引擎，由AIDC-AI团队 desarrollado，能够自动完成写作_video文本、 generar AI 配图、生成AI视频、合成语音解说、添加背景音乐以及一键合成视频。
 
** Features **

*  ेवल门槛：零门槛，-zero-trip-experience，让视频创作成为一句话的事！
*   **Web 界面预览** : 使用 Pixelle-Video自动创建一台视网膜上方的视觉效果界面，为用户提供对视频生成流程的实时预览。
 
** 视频生成流程 **

Pixelle-Video 采用组件化设计，整个视频生成流程清晰简洁：

从输入文本到最终视频输出，整个流程简洁清晰：文案生成 -> 配图规划 -> 逐帧处理 -> 视频合成
 
每个环节都支持灵活定制，可选择不同的AI-modelfonucmodel、音频引擎、视觉风格等，满足个性化创作需求。
