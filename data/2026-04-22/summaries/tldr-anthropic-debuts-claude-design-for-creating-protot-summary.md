# Anthropic Debuts Claude Design for Creating Prototypes, Pitch Decks, and Mockups - MacRumors

## Introducing Claude Design by Anthropic

Anthropic has launched Claude Design, a new AI product designed for creating designs, prototypes, slides, and more. 

### Key Features:

*   Uses Opus 4.7, a new vision model capable of higher-resolution images
*   "More tasteful and creative" in professional tasks compared to previous models
*   Ability to create higher-quality interfaces, slides, and documents with increased resolution
*   Supports collaboration tools for multiple users to access and edit designs simultaneously
*   Allows exporting designs as Canva, PDF, PPTX, and standalone HTML files

### User Benefits:

*   Easy access to design assets from existing brand files and web captures
*   Integration with existing Adobe and Canva products for seamless workflows
*   Support for project import and export of Claude Design projects via various file formats
