# T3 Code

T3 Code is a minimal web GUI for coding agents (currently Codex and Claude, more coming soon).

## Installation

Warning

T3 Code currently supports Codex and Claude.
Install and authenticate at least one provider before use:

* Codex: installCodex CLIand runcodex login
* Claude: install Claude Code and runclaude auth login

### Run without installing

npx t3

### Desktop app

Install the latest version of the desktop app fromGitHub Releases, or from your favorite package registry:

#### Windows (winget)

winget install T3Tools.T3Code

#### macOS (Homebrew)

brew install --cask t3-code

#### Arch Linux (AUR)

yay -S t3code-bin
